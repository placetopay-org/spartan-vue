# Migration Guide: Spartan Vue 2.x to 3.x

Migrating from Spartan Vue 2.x (Tailwind CSS 3) to Spartan Vue 3.x (Tailwind CSS 4).

<div className="alert alert-info">

## Quick Overview

Spartan Vue 3.x introduces full support for Tailwind CSS 4 with a CSS-native configuration approach. The migration requires:

<div className="quick-steps">

### Migration Steps Summary

1. **Remove** Spartan Vue plugin from `tailwind.config.js`
2. **Run** `npx @tailwindcss/upgrade` to migrate to Tailwind 4
3. **Remove** PostCSS configuration (if applicable)
4. **Install** `@tailwindcss/vite` plugin (if using Vite)
5. **Update** to Spartan Vue 3.x: `npm install @placetopay/spartan-vue@latest`
6. **Move** CSS imports from JavaScript to your main CSS file
7. **Update** color imports API (if using Tailwind colors in JS)
8. **Test** your application

</div>

## Key Changes
