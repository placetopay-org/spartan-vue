# Desing System


We have developed an innovative design system called <strong>"Spartan"</strong> that is used in both current and future projects.

Our goal in creating this system was to unify the visual style across all projects. In our design and development process, we relied on the framework called [**Tailwind**](https://tailwindui.com/), which has its own components.

This allowed us to save a lot of time in developing our own design system, as we used this framework as a foundation to create a more customized one.

<strong className="title">Desing</strong>

Our design system comes pre-designed from the [**figma**](https://www.figma.com/) platform, which you can view in a browser or [**install the software for free**](https://www.figma.com/downloads/)
