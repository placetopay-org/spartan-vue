# Desing System

Hemos desarrollado un innovador sistema de diseño llamado  <strong>"Spartan"</strong> que se utiliza tanto en proyectos actuales como en futuros.

Nuestro objetivo al crear este sistema fue unificar la línea gráfica en todos los proyectos. En nuestro proceso de diseño y desarrollo, nos basamos en el framework llamado [**Tailwind**](https://tailwindui.com/), que cuenta con sus propios componentes.

Esto nos permitió ahorrar mucho tiempo en el desarrollo de nuestro propio sistema de diseño, ya que utilizamos este framework como base para crear uno más personalizado.

<strong className="title">Diseño</strong>

Nuestro sistema de diseño viene ya diseñado desde la plataforma [**figma**](https://www.figma.com/) el cual pueden visualizarlo en un navegador o [**instalar el software de manera gratuita**](https://www.figma.com/downloads/)
