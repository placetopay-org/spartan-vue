Spartan includes built-in translations for its components in five languages. For these texts to display correctly, your project must be configured with [vue-i18n](https://vue-i18n.intlify.dev/).

## Supported languages

| Code | Language   |
| ---- | ---------- |
| `es` | Spanish    |
| `en` | English    |
| `it` | Italian    |
| `pt` | Portuguese |
| `fr` | French     |

## Installing vue-i18n

If your project does not yet use vue-i18n, install it:

```bash [Terminal]
npm install vue-i18n
```

## Basic configuration

Import the translations exported by Spartan and incorporate them into your i18n instance:

```ts [main.ts]
import '@placetopay/spartan-vue/style.css'

import { createApp } from 'vue'
import { createI18n } from 'vue-i18n'
import App from './App.vue'
import { es, en } from '@placetopay/spartan-vue/locales'

const i18n = createI18n({
  locale: 'en',
  fallbackLocale: 'es',
  legacy: false,
  messages: {
    es,
    en,
  },
})

const app = createApp(App)
app.use(i18n)
app.mount('#app')
```

::prose-warning
It is important to use `legacy: false` so that vue-i18n works in Composition API mode, which is the mode supported by Spartan components.
::

## Adding your own translations

If you need to combine Spartan's translations with those of your application, merge the messages using spread:

```ts [main.ts]
import { es as spartanEs, en as spartanEn } from '@placetopay/spartan-vue/locales'

const messages = {
  es: {
    ...spartanEs,
    // your custom translations
    app: {
      title: 'Mi Aplicación',
      welcome: 'Bienvenido',
    },
  },
  en: {
    ...spartanEn,
    app: {
      title: 'My Application',
      welcome: 'Welcome',
    },
  },
}
```

::prose-tip
Spartan's translations use their own internal namespace, so conflicts with your translation keys are unlikely.
::

## Dynamic language switching

With vue-i18n in Composition API mode you can change the language at runtime:

```vue [LanguageSwitcher.vue]
<script setup>
import { useI18n } from 'vue-i18n'

const { locale } = useI18n()

const languages = [
  { code: 'es', name: 'Español' },
  { code: 'en', name: 'English' },
  { code: 'fr', name: 'Français' },
]
</script>

<template>
  <div>
    <button
      v-for="language in languages"
      :key="language.code"
      @click="locale = language.code"
    >
      {{ language.name }}
    </button>
  </div>
</template>
```
