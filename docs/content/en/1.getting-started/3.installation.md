## Requirements

Before installing Spartan, make sure you meet the following requirements:

- **Node.js** `20.19` or `^22.12`
- **NPM** `^10`
- **TailwindCSS** `^4` configured in your project
- An existing **Vue 3** project

## Step-by-step installation

::steps
### Install the package

Run the following command in the root directory of your project:

::::code-group

```bash [npm]
npm install -D @placetopay/spartan-vue
```

```bash [pnpm]
pnpm add -D @placetopay/spartan-vue
```

```bash [yarn]
yarn add --dev @placetopay/spartan-vue
```
::::

### Configure TailwindCSS v4

In TailwindCSS v4 the configuration is done from CSS, there is no `tailwind.config.js`. Add the Spartan `@import` and the `@source` directive in your main stylesheet:

```css [styles.css]
@import "@placetopay/spartan-vue/styles.css";

@source '../src/**/*.{vue,js,ts}';
```

::prose-tip
The Spartan `@import` must be **before** any other custom styles. The `@source` directive tells Tailwind which files in your project to scan to generate utility classes.
::

### Import the styles in your entry point

Import your stylesheet in the application entry point:

```ts [main.ts]
import './styles.css'

import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)
app.mount('#app')
```

### Use the components

The components are ready to use. Import them directly in your `.vue` files:

```vue [App.vue]
<script setup>
import { SButton } from '@placetopay/spartan-vue'
</script>

<template>
  <SButton>Create</SButton>
</template>
```
::

## Theme customization (optional)

By default, Spartan includes a primary color palette. In TailwindCSS v4 the customization is done with CSS variables inside `@theme`, placed **after** the Spartan `@import`:

```css [styles.css]
@import "@placetopay/spartan-vue/styles.css";

@source '../src/**/*.{vue,js,ts}';

@theme {
  --color-spartan-primary-50:  rgb(228 242 253);
  --color-spartan-primary-100: rgb(187 222 251);
  --color-spartan-primary-200: rgb(144 202 249);
  --color-spartan-primary-300: rgb(100 181 246);
  --color-spartan-primary-400: rgb(66 165 245);
  --color-spartan-primary-500: rgb(33 150 243);
  --color-spartan-primary-600: rgb(30 132 229);
  --color-spartan-primary-700: rgb(25 103 196);
  --color-spartan-primary-800: rgb(21 78 148);
  --color-spartan-primary-900: rgb(13 42 84);
}
```

### Excluding the Inter font

Spartan includes the Inter font by default. If you are already using another typeface, import the style files separately instead of the main `@import`:

```css [styles.css]
@import "tailwindcss";
@import "@placetopay/spartan-vue/styles/plugin.css";
@import "@placetopay/spartan-vue/styles/spartan-vue.css";

@source '../src/**/*.{vue,js,ts}';
```
