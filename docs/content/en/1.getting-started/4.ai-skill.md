The **Spartan Vue skill** gives your AI assistant complete knowledge of the library's 68 components: what each one does, what props it accepts, what variants it has, how to customize it, and how to combine them to build complete interfaces.

## Installation

Run the following command from your project root:

```bash
npx skills add placetopay-org/spartan-vue --skill spartan-vue
```

## Example tasks

With the skill installed, you can ask the assistant things like:

- **"I need a login form"** — Combines `SInput`, `SInputPassword`, `SButton`, and `SCheckbox` with the right props.
- **"Show me all SAlert variants"** — Generates code with all 7 color palettes and 4 styles (solid, outline, soft, subtle).
- **"I want a data table with pagination"** — Uses `SDTable` with `@tanstack/vue-table` configuration and `SPaginator`.
- **"How do I customize the SCard header?"** — Explains `pt:title` and `pt:description` usage with passthrough syntax.
- **"Set up vue-i18n so components show in Spanish"** — Full setup with locales and helpers.
- **"Change the primary color of the whole library to blue"** — Overrides the 10 `--color-spartan-primary-*` variables with `@theme`.

## What does the skill know?

| Area | Detail |
|------|--------|
| **Component API** | Props, emits, slots, and sub-components for all 68 components |
| **Categories** | Data input, selectors, display, modals, structure, utilities, typography |
| **Block variants** | Wrappers like `SInputBlock`, `SSelectBlock` that add label, help, and error UI |
| **Passthrough** | `usePassthrough()` keys per component (16 supported components) |
| **Themes** | CSS variables `--color-spartan-primary-*`, outline/ring utilities, Inter exclusion |
| **i18n** | `vue-i18n` setup, merge helpers, 5 languages, `$spartan.*` keys per component |
| **Installation** | Full setup: npm install, TailwindCSS v4, imports, troubleshooting |
| **Patterns** | v-model, polymorphic `as` prop, form states, dark mode, icons |

::prose-tip
The skill works best inside projects that use `@placetopay/spartan-vue`. It automatically provides context so the assistant generates code that follows the library's conventions.
::
