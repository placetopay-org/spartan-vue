# Usage

The `SButtonGroup` component renders a group of connected buttons. Use `SButtonGroupItem` children to define each button. Set `first` and `last` on the edge items for proper border rounding.

::component-preview{file="SButtonGroup/basic"}
::

## With Icons

Use the `icon` prop on `SButtonGroupItem` to display an icon alongside the button text.

::component-preview{file="SButtonGroup/with-icons"}

Use the `prev` and `next` props to render arrow navigation buttons.

## Props

### SButtonGroup

The container component has no props. It renders a `<div>` with inline flex layout.

### SButtonGroupItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `active` | `boolean` | `false` | Highlights the button as active |
| `disabled` | `boolean` | `false` | Disables the button |
| `first` | `boolean` | `false` | Rounds the left border |
| `last` | `boolean` | `false` | Rounds the right border |
| `prev` | `boolean` | `false` | Renders a left arrow navigation button |
| `next` | `boolean` | `false` | Renders a right arrow navigation button |
| `icon` | `FunctionalComponent` | - | Icon component to display |
| `endIcon` | `boolean` | `false` | Places the icon after the text |
| `as` | `string \| Component` | `'button'` | The element or component to render as |
| `class` | `string` | - | Additional CSS classes |
