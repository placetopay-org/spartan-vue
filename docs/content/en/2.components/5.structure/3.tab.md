# Usage

`STab` is a compound tab navigation component. Use `STabItem` inside `STab` to define each tab. The active tab is controlled via `v-model`, which binds to the `path` prop of each `STabItem`.

::component-preview{file="STab/basic"}
::

## Variants

The `variant` prop changes the visual style of the tab navigation. Available variants are `underline` (bottom border indicator), `pills` (rounded background), and `vetches` (contained rounded with border).

::

## STab Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The active tab path. Required. Use with `v-model`. |
| `variant` | `'underline' \| 'pills' \| 'vetches'` | `'underline'` | Visual style of the tab navigation. |
| `full` | `boolean` | `false` | Makes the tabs stretch to fill the full width. |
| `dropdownResponsive` | `boolean` | `false` | Enables responsive dropdown behavior for overflow tabs. |
| `longestCommonPrefix` | `boolean` | `false` | Uses longest common prefix matching for active tab detection. |
| `class` | `string` | - | Additional CSS classes for the root element. |

## STabItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `path` | `string` | - | Unique identifier for the tab. Used to match the `modelValue`. |
| `as` | `string \| Component` | `'button'` | The HTML element or Vue component to render the tab as. |
| `icon` | `FunctionalComponent` | - | Icon component displayed before the tab label. |
| `active` | `boolean` | - | Manually sets the tab as active, overriding path matching. |
| `regex` | `RegExp` | - | Regular expression for custom active state matching against the model value. |
| `dropdown` | `boolean` | `false` | Renders the tab as a dropdown trigger. Use the `items` slot for dropdown content. |
| `class` | `string` | - | Additional CSS classes for the tab item. |

## STabDropdownItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `path` | `string` | - | Unique identifier for the dropdown item. Used to match the `modelValue`. |
| `regex` | `RegExp` | - | Regular expression for custom active state matching. |
| `as` | `string \| Component` | - | The HTML element or Vue component to render as. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the active tab changes. |

## Slots

### STab

| Name | Description |
|------|-------------|
| `default` | Tab items (`STabItem` components). |

### STabItem

| Name | Description |
|------|-------------|
| `default` | Tab label content. |
| `items` | Dropdown menu content when `dropdown` is `true`. |
