# Usage

The `STemplateHeaderTable` component renders a header with a title and search input. Use the `table` slot to place your table content below.

::component-preview{file="STemplateHeaderTable/basic"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | The heading text displayed in the header |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `search` | `string` | Emitted when the user clicks the search button or presses Enter |

## Slots

| Name | Description |
|------|-------------|
| `table` | The table content to display below the header |
