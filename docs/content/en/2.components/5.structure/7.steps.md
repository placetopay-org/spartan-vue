# Usage

The `SSteps` component renders a step-by-step progress indicator. Place `SStepsItem` children inside with `status` set to `complete`, `current`, or `upcoming`.

::component-preview{file="SSteps/basic"}
::

## Variants

Use the `variant` prop to switch between `circlesWithText` (default) and `simple` styles.

::component-preview{file="SSteps/variants"}
::

## Steps Array

Alternatively, pass an array of step objects via the `steps` prop instead of using `SStepsItem` children.

::component-preview{file="SSteps/array"}
::

## Props

### SSteps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'circlesWithText' \| 'simple'` | `'circlesWithText'` | The visual style variant |
| `steps` | `TStepsItemsProps[]` | `[]` | Array of step objects (alternative to slot usage) |

### SStepsItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `status` | `'complete' \| 'current' \| 'upcoming'` | - | The step's current status |
| `href` | `string` | - | The link URL for the step |
| `name` | `string` | - | The step title text |
| `description` | `string` | - | The step description text |
| `last` | `boolean` | `false` | Marks the last step (hides the connecting line) |

## Slots

### SSteps

| Name | Description |
|------|-------------|
| `default` | `SStepsItem` components defining each step |

### SStepsItem

| Name | Description |
|------|-------------|
| `default` | Custom step name content |
| `description` | Custom step description content |
