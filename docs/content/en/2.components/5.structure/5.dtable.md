# Usage

`SDTable` automatically generates a table from a `data` array and `SDColumn` definitions. Each `SDColumn` maps to a column in the table by matching its `field` prop to a key in the data objects. It supports column sorting, loading skeletons, row links, expandable rows, custom cell rendering, and pagination.

::component-preview{file="SDTable/basic"}
::

## Loading

Set the `loading` prop to `true` to display a skeleton placeholder while data is being fetched. The skeleton automatically matches the number of defined columns.

::component-preview{file="SDTable/loading"}
::

## Sorting

Add the `sort` prop to any `SDColumn` to make that column sortable. Clicking the column header cycles through sort states. The `sort` event is emitted with the column `field` and current `sort` direction.

::

## SDTable Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `Record<string, any>[]` | - | Array of row objects to display in the table |
| `loading` | `boolean` | `false` | Shows a skeleton loading state |
| `slim` | `boolean` | `false` | Reduces cell padding for a more compact layout |
| `borderless` | `boolean` | `false` | Removes the border and shadow around the table |
| `rowLink` | `(row: Record<string, any>) => string` | - | Function that returns a URL to make each row a navigable link |
| `rowLinkAs` | `Component` | - | Custom component to render row links (e.g., `RouterLink`) |
| `paginator` | `TPaginatorProps` | - | Paginator configuration object to display pagination controls |
| `class` | `string` | - | Additional CSS classes applied to the table container |

## SDColumn Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `field` | `string` | - | Key in the data object to display in this column |
| `header` | `string` | - | Text displayed in the column header |
| `sort` | `boolean \| 'asc' \| 'des'` | - | Enables sorting on this column. Set to `true` for default, or `'asc'`/`'des'` for an initial direction |
| `noLink` | `boolean` | - | Prevents this column from being wrapped in a row link |
| `expander` | `boolean` | - | Makes this column an expander toggle for expandable rows |
| `unstyled` | `boolean` | - | Removes default cell padding and text styles |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `sort` | `{ field: string \| symbol, sort: 'asc' \| 'des' \| boolean }` | Emitted when a sortable column header is clicked |
| `paginatorChange` | `{ page?: number, size?: number, dir?: 'prev' \| 'next' }` | Emitted when the user interacts with pagination controls |

## SDColumn Slots

| Name | Props | Description |
|------|-------|-------------|
| `body` | `{ row: any, value: any }` | Custom cell content. `row` is the full data object, `value` is the field value |
| `header` | - | Custom header content for the column |
