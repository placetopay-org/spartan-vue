# Usage

The `SStackedList` component renders a styled list with dividers. Use `SStackedListItem` children to define each item.

::component-preview{file="SStackedList/basic"}
::

## Dropdown Menu

Use the `dropdown` slot on `SStackedListItem` to add an options menu with `SDropdownItem` elements.

::component-preview{file="SStackedList/dropdown"}
::

## Props

### SStackedList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes for the list container |

### SStackedListItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes for the item content |

## Slots

### SStackedList

| Name | Description |
|------|-------------|
| `default` | `SStackedListItem` components |

### SStackedListItem

| Name | Description |
|------|-------------|
| `default` | The item content |
| `dropdown` | Dropdown menu items (renders an options menu icon) |
