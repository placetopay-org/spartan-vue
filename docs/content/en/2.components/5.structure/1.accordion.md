# Usage

The `SAccordion` component provides smooth expand and collapse animations using CSS grid transitions. It wraps content and animates its visibility based on the `open` prop. By default it animates horizontally; use `vertical` for the typical top-to-bottom accordion behavior.

::component-preview{file="SAccordion/basic"}
::

## Vertical

When the `vertical` prop is set, the content expands and collapses along the vertical axis using `grid-template-rows`. This is the most common pattern for accordion-style UI.

::component-preview{file="SAccordion/vertical"}
::

## Horizontal

Without the `vertical` prop, the content expands and collapses along the horizontal axis using `grid-template-columns`. Useful for side panels or reveal animations.

::component-preview{file="SAccordion/horizontal"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controls whether the content is expanded or collapsed. Required. |
| `vertical` | `boolean` | `false` | When `true`, animates vertically (rows). When `false`, animates horizontally (columns). |
| `class` | `string` | - | Custom CSS classes applied to the root element. |

## Slots

| Name | Description |
|------|-------------|
| `default` | The content to be expanded or collapsed. |
