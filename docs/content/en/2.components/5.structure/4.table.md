# Usage

`STable` is a compound table component built from composable sub-components: `STableHead`, `STableBody`, `STableRow`, `STableCell`, and `STableHeadCell`. Each sub-component maps to a native HTML table element (`<table>`, `<thead>`, `<tbody>`, `<tr>`, `<td>`, `<th>`) with consistent styling applied automatically.

You can also auto-generate a table by passing `cols` and `rows` arrays directly to `STable`, skipping the sub-components entirely.

::

## Borderless

By default, the table renders with a rounded border, shadow, and outline. Set `borderless` to `true` to remove the outer container styling, which is useful when embedding the table inside another bordered container.

## Props

### STable

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `ClassNameValue` | - | Additional CSS classes applied to the table |
| `borderless` | `boolean` | `false` | Removes the outer border, shadow, and rounded corners |
| `cols` | `string[]` | - | Column headers for auto-generated table |
| `rows` | `string[]` | - | Row data for auto-generated table |
| `highlight` | `number[]` | - | Column indices to highlight in auto-generated rows |

### STableHead

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes applied to the thead |

### STableBody

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes applied to the tbody |

### STableRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `ClassNameValue` | - | Additional CSS classes applied to the row |

### STableHeadCell

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `ClassNameValue` | - | Additional CSS classes applied to the header cell |

### STableCell

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `ClassNameValue` | - | Additional CSS classes applied to the cell |
| `highlight` | `boolean` | `false` | Applies bold font weight and darker text color |
| `colspan` | `number` | - | Number of columns the cell should span |

## Slots

### STable

| Name | Description |
|------|-------------|
| `default` | Table content (STableHead, STableBody, or other elements) |

### STableHead

| Name | Description |
|------|-------------|
| `default` | Header cells (STableHeadCell elements) |

### STableBody

| Name | Description |
|------|-------------|
| `default` | Table rows (STableRow elements) |

### STableRow

| Name | Description |
|------|-------------|
| `default` | Row cells (STableCell or STableHeadCell elements) |

### STableCell

| Name | Description |
|------|-------------|
| `default` | Cell content |

### STableHeadCell

| Name | Description |
|------|-------------|
| `default` | Header cell content |
