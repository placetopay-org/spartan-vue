# Usage

`SSidebar` is a compound navigation component that manages active item state via `v-model`. It supports collapsible groups (`SSidebarItemGroup`), section dividers (`SSidebarSeparator`), and individual navigation items (`SSidebarItem`). Use `placetopayHeader` to show the default PlaceToPay logo, or provide a custom `header` slot.

::component-preview{file="SSidebar/basic"}
::

## Groups

Use `SSidebarItemGroup` to create collapsible sections. By default, groups use accordion behavior (`accordion` is `true`), which wraps child items in an `SAccordion`. Set `accordion` to `false` to render a plain list that is always visible.

::component-preview{file="SSidebar/groups"}
::

## Separator

Use `SSidebarSeparator` to add labeled horizontal dividers between sidebar sections. The `title` prop sets the section heading text.

::component-preview{file="SSidebar/separator"}
::

## SSidebar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The active path value, bound with `v-model` |
| `class` | `string` | - | Additional CSS classes for the sidebar container |
| `placetopayHeader` | `boolean \| (() => any)` | `false` | Shows the PlaceToPay logo header. Pass a function to make the header clickable |
| `nested` | `boolean` | - | Marks the sidebar as nested (used internally) |

## SSidebarItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `string \| Component` | `'button'` | The element or component to render as |
| `path` | `string` | - | Unique path identifier for the item. Defaults to the item's text content |
| `icon` | `FunctionalComponent` | - | Icon component displayed before the label |
| `active` | `boolean` | `false` | Forces the item to display as active |

## SSidebarItemGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `string \| Component` | `'button'` | The element or component to render as |
| `path` | `string` | - | Unique path identifier for the group. Defaults to the title slot text |
| `icon` | `FunctionalComponent` | - | Icon component displayed before the group title |
| `active` | `boolean` | `false` | Forces the group to display as active |
| `accordion` | `boolean` | `true` | When true, children are wrapped in an accordion that collapses/expands on click |

## SSidebarSeparator Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Section heading text displayed below the horizontal rule |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the active path changes |

## Slots

### SSidebar Slots

| Name | Description |
|------|-------------|
| `default` | Navigation items, groups, and separators |
| `header` | Custom header content (replaces the PlaceToPay logo) |
| `footer` | Footer content pinned to the bottom of the sidebar |

### SSidebarItemGroup Slots

| Name | Description |
|------|-------------|
| `default` | Child `SSidebarItem` components |
| `title` | Group title text |
