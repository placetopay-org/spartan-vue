# Usage

The SToast component displays notification messages with automatic icon selection based on the type. It supports titles, descriptions, and can be made closeable.

::spartan-component
---
items: { type: ['success', 'error', 'warning'] }
props:
  type: success
  title: Operation completed
  description: Your changes have been saved successfully
  closeable: false
  leftBorder: false
ignore:
  - class
---
::

## Types

SToast supports three notification types, each with a distinct icon and color scheme: success (green), error (red), and warning (yellow).

::spartan-component
---
items: { type: ['success', 'error', 'warning'] }
props:
  type: success
  title: Notification message
ignore:
  - description
  - closeable
  - leftBorder
  - class
---
::

## With Description

Add a description below the title to provide additional context for the notification.

::spartan-component
---
props:
  type: success
  title: Profile updated
  description: Your profile information has been updated successfully
ignore:
  - closeable
  - leftBorder
  - class
---
::

## Left Border

Enable a colored left border to make the toast more visually prominent.

::spartan-component
---
props:
  type: success
  title: Changes saved
  leftBorder: true
ignore:
  - description
  - closeable
  - class
---
::

## Closeable

Make the toast closeable by adding a close button that emits a `closeToast` event when clicked.

::spartan-component
---
props:
  type: success
  title: Dismissible notification
  description: Click the X button to close this toast
  closeable: true
ignore:
  - leftBorder
  - class
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'success' \| 'error' \| 'warning'` | `'success'` | The notification type that determines icon and colors |
| `title` | `string` | - | The main message text to display |
| `description` | `string` | - | Optional secondary text displayed below the title |
| `closeable` | `boolean` | `false` | Whether to show a close button |
| `leftBorder` | `boolean` | `false` | Whether to display a colored left border |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `closeToast` | - | Emitted when the close button is clicked |

## Slots

| Name | Description |
|------|-------------|
| `default` | Custom content for the title (replaces the `title` prop) |
| `description` | Custom content for the description (replaces the `description` prop) |
