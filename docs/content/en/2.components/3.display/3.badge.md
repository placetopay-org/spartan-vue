# Usage

The `SBadge` component displays a compact inline label. Use the default slot to set the badge content.

::component-preview{file="SBadge/basic"}
::

## Colors

Use the `color` prop to change the badge's color.

::component-preview{file="SBadge/colors"}
::

## Sizes

Use the `size` prop to change the badge size.

::component-preview{file="SBadge/sizes"}
::

## Outline

Set `outline` to `true` to display an outlined style with a lighter background.

::component-preview{file="SBadge/outline"}
::

## Pill

Set `pill` to `true` to use fully rounded corners.

::component-preview{file="SBadge/pill"}
::

## Dot

Set `dot` to `true` to show a colored dot indicator before the badge content.

::component-preview{file="SBadge/dot"}
::

## Removable

Set `removable` to `true` to show a remove button. The component emits a `removed` event when clicked. Use `'stopPropagation'` to also stop event propagation.

::component-preview{file="SBadge/removable"}
::

## Tag Slot

Use the `tag` slot to display a highlighted label section inside the badge.

::component-preview{file="SBadge/tag"}
::

## Reverse

Set `reverse` to `true` to reverse the order of the tag and content.

::component-preview{file="SBadge/reverse"}
::

## Combined

Combine multiple props for different styles.

```vue
<template>
  <SBadge color="red" :pill="true" :dot="true" :outline="true">Critical</SBadge>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `'gray' \| 'white' \| 'primary' \| 'red' \| 'blue' \| 'green' \| 'yellow' \| 'indigo' \| 'purple' \| 'neutral'` | `'gray'` | Badge color |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Badge size |
| `dot` | `boolean` | `false` | Shows a colored dot indicator |
| `outline` | `boolean` | `false` | Applies outlined style |
| `pill` | `boolean` | `false` | Applies fully rounded corners |
| `removable` | `boolean \| 'stopPropagation'` | `false` | Shows a remove button. Use `'stopPropagation'` to stop event propagation on remove |
| `reverse` | `boolean` | `false` | Reverses the order of tag and content |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `removed` | - | Emitted when the remove button is clicked |

## Slots

| Name | Description |
|------|-------------|
| `default` | Badge content |
| `tag` | Highlighted label section inside the badge |
