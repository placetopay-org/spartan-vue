# Usage

The SBreadcrumbs component works as a container for SBreadcrumbsItem components. Each item represents a level in the navigation hierarchy.

::spartan-component
---
slug: breadcrumbs-demo
external: ['items']
props:
  items:
    - label: Home
      href: /
      active: false
    - label: Products
      href: /products
      active: false
    - label: Laptops
      href: /products/laptops
      active: true
ignore:
  - class
---
::

## Active State

The last breadcrumb item should have `active` set to `true` to indicate the current page. Active items are styled with darker text.

::spartan-component
---
slug: breadcrumbs-demo
external: ['items']
props:
  items:
    - label: Home
      href: /
      active: false
    - label: Settings
      href: /settings
      active: true
ignore:
  - class
---
::

## Code Example

Here's how to use SBreadcrumbs with SBreadcrumbsItem in your code:

```vue
<template>
  <SBreadcrumbs>
    <SBreadcrumbsItem :first="true" href="/" :active="false">
      Home
    </SBreadcrumbsItem>
    <SBreadcrumbsItem :first="false" href="/products" :active="false">
      Products
    </SBreadcrumbsItem>
    <SBreadcrumbsItem :first="false" href="/products/laptops" :active="true">
      Laptops
    </SBreadcrumbsItem>
  </SBreadcrumbs>
</template>
```

## SBreadcrumbs Slots

| Name | Description |
|------|-------------|
| `default` | Container for SBreadcrumbsItem components |

## SBreadcrumbsItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `first` | `boolean` | - | Whether this is the first item (hides the separator) |
| `href` | `string` | - | The URL this breadcrumb item links to |
| `active` | `boolean` | - | Whether this item represents the current page |
| `icon` | `FunctionalComponent` | - | Optional icon to display before the text |
| `as` | `string \| Component` | `'a'` | The component or element to render the link as |

## SBreadcrumbsItem Slots

| Name | Description |
|------|-------------|
| `default` | The text content of the breadcrumb item |
| `separator` | Custom separator element to display before this item (not shown when `first` is true) |
