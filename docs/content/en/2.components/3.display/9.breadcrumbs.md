# Usage

The `SBreadcrumbs` component works as a container for `SBreadcrumbsItem` components. Each item represents a level in the navigation hierarchy.

::component-preview{file="SBreadcrumbs/basic"}
::

## Active State

The last breadcrumb item should have `active` set to `true` to indicate the current page. Active items are styled with darker text and set `aria-current="page"` for accessibility.

::component-preview{file="SBreadcrumbs/active"}
::

## First Item

Set `first` to `true` on the first item to hide the chevron separator.

::component-preview{file="SBreadcrumbs/first-item"}
::

## With Icon

Use the `icon` prop to display an icon before the breadcrumb text.

::component-preview{file="SBreadcrumbs/with-icon"}
::

## Custom Separator

Use the `separator` slot on `SBreadcrumbsItem` to replace the default chevron separator.

::component-preview{file="SBreadcrumbs/custom-separator"}
::

## Custom Element

Use the `as` prop to render the breadcrumb item as a different element or component, useful for router integration.

::component-preview{file="SBreadcrumbs/custom-element"}
::

## SBreadcrumbs Slots

| Name | Description |
|------|-------------|
| `default` | Container for `SBreadcrumbsItem` components |

## SBreadcrumbsItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `first` | `boolean` | `false` | Whether this is the first item (hides the separator) |
| `href` | `string` | - | The URL this breadcrumb item links to |
| `active` | `boolean` | `false` | Whether this item represents the current page |
| `icon` | `FunctionalComponent` | - | Optional icon to display before the text |
| `as` | `string \| Component` | `'a'` | The component or element to render the link as |

## SBreadcrumbsItem Slots

| Name | Description |
|------|-------------|
| `default` | The text content of the breadcrumb item |
| `separator` | Custom separator element (replaces the default chevron, not shown when `first` is true) |
