# Usage

The `SSkeleton` component provides an animated placeholder to indicate content is loading. It renders a shimmer animation and can be customized via the `class` prop.

::component-preview{file="SSkeleton/basic"}
::

## Sizes

Customize the height and width using Tailwind classes.

```vue
<template>
  <SSkeleton class="h-4 w-3/4 rounded-md" />
</template>
```

```vue
<template>
  <SSkeleton class="h-8 w-1/2 rounded-md" />
</template>
```

```vue
<template>
  <SSkeleton class="h-12 w-full rounded-lg" />
</template>
```

## Shapes

### Circle

```vue
<template>
  <SSkeleton class="h-12 w-12 rounded-full" />
</template>
```

### Square

```vue
<template>
  <SSkeleton class="h-12 w-12 rounded-md" />
</template>
```

## Composition

Combine multiple skeletons to build loading placeholders for complex layouts.

```vue
<template>
  <div class="flex items-center gap-4">
    <SSkeleton class="h-12 w-12 rounded-full" />
    <div class="flex-1 space-y-2">
      <SSkeleton class="h-4 w-3/4" />
      <SSkeleton class="h-4 w-1/2" />
    </div>
  </div>
</template>
```

```vue
<template>
  <div class="space-y-3">
    <SSkeleton class="h-40 w-full rounded-lg" />
    <SSkeleton class="h-4 w-3/4" />
    <SSkeleton class="h-4 w-1/2" />
    <SSkeleton class="h-4 w-5/6" />
  </div>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | CSS classes to customize size, shape, and border radius. Merged with default classes (`h-6 w-full rounded-md`) |
