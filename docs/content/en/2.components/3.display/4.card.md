# Usage

The `SCard` component provides a flexible container for content. It supports titles, descriptions, icons, action buttons, and a closable state.

::spartan-component
---
props:
  title: Card Title
  size: md
slots:
  default: This is the card content. You can place any content here.
ignore:
  - icon
  - iconColor
  - iconType
  - closable
  - actions
  - class
---
::

## Sizes

Use the `size` prop to change the card's padding and border radius.

::spartan-component
---
items: { size: ['sm', 'md'] }
props:
  title: Card Title
  size: md
slots:
  default: Card content with adjustable size.
ignore:
  - icon
  - iconColor
  - iconType
  - closable
  - actions
  - class
---
::

## With Title and Description

Use the `title` prop or the `title` slot, and the `description` slot for additional text.

::spartan-component
---
props:
  title: Payment Successful
slots:
  description: Your transaction has been processed successfully.
  default: ''
ignore:
  - size
  - icon
  - iconColor
  - iconType
  - closable
  - actions
  - class
---
::

## With Icon

Use the `icon` prop with a preset string (`'primary'`, `'success'`, `'danger'`, `'warning'`, `'info'`) to display a predefined icon with matching colors.

::spartan-component
---
items: { icon: ['primary', 'success', 'danger', 'warning', 'info'] }
props:
  icon: success
  title: Operation Complete
slots:
  description: Everything went as expected.
  default: ''
ignore:
  - size
  - iconColor
  - iconType
  - closable
  - actions
  - class
---
::

## Icon Type

Use the `iconType` prop to switch between `'solid'` (default) and `'ping'` styles.

::spartan-component
---
items: { icon: ['primary', 'success', 'danger', 'warning', 'info'], iconType: ['solid', 'ping'] }
props:
  icon: primary
  iconType: solid
  title: Notification
slots:
  description: You have a new message.
  default: ''
ignore:
  - size
  - iconColor
  - closable
  - actions
  - class
---
::

## Icon Color Override

Use the `iconColor` prop to override the icon's background and text color independently of the icon variant.

::spartan-component
---
items: { icon: ['primary', 'success', 'danger', 'warning', 'info'], iconColor: ['primary', 'success', 'danger', 'warning', 'info'] }
props:
  icon: primary
  iconColor: danger
  title: Custom Icon Color
slots:
  description: The icon shape is primary but colored as danger.
  default: ''
ignore:
  - size
  - iconType
  - closable
  - actions
  - class
---
::

## Custom Icon

Pass a Vue component to the `icon` prop to use a custom icon.

```vue
<template>
  <SCard :icon="ShieldTickIcon" title="Secured">
    <template #description>Your account is protected.</template>
  </SCard>
</template>

<script setup lang="ts">
import { ShieldTickIcon } from '@placetopay/iconsax-vue/outline';
</script>
```

## Closable

Set `closable` to `true` to display a close button. The component emits a `close` event when clicked.

::spartan-component
---
props:
  icon: warning
  title: Session Expiring
  closable: true
slots:
  description: Your session will expire in 5 minutes.
  default: ''
hide:
  - closable
ignore:
  - size
  - iconColor
  - iconType
  - actions
  - class
---
::

## Actions Slot

Use the `actions` slot to add custom action buttons at the bottom of the card.

```vue
<template>
  <SCard icon="success" title="Invitation Sent">
    <template #description>The invitation has been sent to the user.</template>
    <template #actions>
      <SButton variant="primary">Accept</SButton>
      <SButton variant="secondary">Decline</SButton>
    </template>
  </SCard>
</template>
```

## Actions Prop

Use the `actions` prop to render action buttons with icons in a divided footer section.

```vue
<template>
  <SCard
    title="Quick Actions"
    :actions="[
      { icon: PencilIcon, text: 'Edit', onClick: () => console.log('Edit') },
      { icon: TrashIcon, text: 'Delete', onClick: () => console.log('Delete') },
    ]"
  >
    <template #description>Choose an action below.</template>
  </SCard>
</template>

<script setup lang="ts">
import { PencilIcon, TrashIcon } from '@heroicons/vue/24/outline';
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Card title text |
| `size` | `'sm' \| 'md'` | `'md'` | Card size (affects padding and border radius) |
| `icon` | `FunctionalComponent \| 'primary' \| 'success' \| 'danger' \| 'warning' \| 'info'` | - | Icon to display. Use a string for presets or a component for custom icons |
| `iconColor` | `'primary' \| 'success' \| 'danger' \| 'warning' \| 'info'` | - | Overrides the icon color independently of the icon variant |
| `iconType` | `'solid' \| 'ping'` | `'solid'` | Icon display style |
| `closable` | `boolean` | - | Shows a close button in the header |
| `actions` | `TAction[]` | - | Array of action buttons rendered in a divided footer |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `close` | - | Emitted when the close button is clicked |

## Slots

| Name | Description |
|------|-------------|
| `default` | Main card content |
| `title` | Custom title content (overrides `title` prop) |
| `description` | Description text below the title |
| `actions` | Action buttons section at the bottom |
