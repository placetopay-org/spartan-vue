# Usage

The SLink component renders a styled anchor tag with built-in hover effects and underline behavior.

::component-preview{file="SLink/basic"}
::

## Target

The `target` prop controls where the linked document will open. Use `_blank` to open links in a new tab, which is common for external links.

::component-preview{file="SLink/target"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | The URL the link points to (required) |
| `target` | `'_blank' \| '_self' \| '_parent' \| '_top'` | - | Specifies where to open the linked document |

## Slots

| Name | Description |
|------|-------------|
| `default` | The clickable text content of the link |
