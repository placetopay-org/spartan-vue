# Usage

The `SAlert` component displays important messages with support for multiple color themes, visual variants, icons, and a closeable option.

::spartan-component
---
props:
  color: info
  variant: soft
  title: Heads up!
  description: This is an informational alert message.
  closeable: false
ignore:
  - icon
  - closeIcon
  - class
---
::

## Colors

Use the `color` prop to change the alert's color theme.

::spartan-component
---
items: { color: ['neutral', 'primary', 'secondary', 'success', 'info', 'warning', 'error'] }
props:
  color: primary
  variant: solid
  title: Alert title
  description: This is an alert description.
ignore:
  - closeable
  - icon
  - closeIcon
  - class
---
::

## With Slot Content

When no `description` prop is provided, the default slot is used as the alert content.

```vue
<template>
  <SAlert color="info" variant="subtle" title="Note">
    <p>This is <strong>rich content</strong> passed via the default slot.</p>
  </SAlert>
</template>
```

## Closeable

Set `closeable` to `true` to show a close button. The component emits a `close` event when clicked.

::spartan-component
---
props:
  color: error
  variant: soft
  title: Error
  description: Something went wrong. Please try again.
  closeable: true
ignore:
  - icon
  - closeIcon
  - class
---
::

## With Icon

Use the `icon` prop to display an icon alongside the alert content.

```vue
<template>
  <SAlert
    color="info"
    variant="subtle"
    title="Information"
    description="This alert includes an icon."
    :icon="InfoCircleIcon"
  />
</template>

<script setup lang="ts">
import { InfoCircleIcon } from '@placetopay/iconsax-vue/outline';
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Title text displayed at the top of the alert |
| `description` | `string` | - | Descriptive text content of the alert |
| `icon` | `FunctionalComponent` | - | Icon component displayed alongside the content |
| `color` | `'neutral' \| 'primary' \| 'secondary' \| 'success' \| 'info' \| 'warning' \| 'error'` | `'neutral'` | Color theme of the alert |
| `variant` | `'solid' \| 'outline' \| 'soft' \| 'subtle'` | `'solid'` | Visual style variant |
| `closeable` | `boolean` | `false` | Shows a close button |
| `closeIcon` | `FunctionalComponent` | `XMarkIcon` | Custom icon for the close button |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `close` | - | Emitted when the close button is clicked |

## Slots

| Name | Description |
|------|-------------|
| `default` | Alert content. Used when `description` prop is not provided |
