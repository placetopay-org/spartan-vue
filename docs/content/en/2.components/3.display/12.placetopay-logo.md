# Usage

The SPlacetopayLogo component renders the Placetopay brand logo with support for different visual modes and customizable dimensions.

::component-preview{file="SPlacetopayLogo/basic"}
::

## Modes

Choose between three color modes depending on your background: base (gray), dark (white), or blackAndWhite.

::component-preview{file="SPlacetopayLogo/modes"}
::

## Predefined Size

Use the `size` prop for quick sizing. The "md" size sets the width to 202px automatically while maintaining the aspect ratio.

::component-preview{file="SPlacetopayLogo/size"}
::

## Custom Dimensions

Set custom dimensions using the `width` and `height` props. The component maintains a 202:45 aspect ratio by default.

::component-preview{file="SPlacetopayLogo/custom-dimensions"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `'base' \| 'dark' \| 'blackAndWhite'` | `'base'` | The color mode of the logo |
| `size` | `'md'` | - | Predefined size (md = 202px width) |
| `width` | `number` | - | Custom width in pixels (overrides size) |
| `height` | `number` | - | Custom height in pixels |
| `class` | `string` | - | Additional CSS classes |
