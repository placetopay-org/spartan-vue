# Usage

The `SAvatar` component displays a user avatar with dark mode support. It shows initials derived from the `name` prop, or an image when `src` is provided.

::component-preview{file="SAvatar/basic"}
::

## Sizes

Use the `size` prop to change the avatar size.

::component-preview{file="SAvatar/sizes"}
::

## Initials

The component automatically generates initials from the `name` prop. It supports names separated by spaces, hyphens, dots, and other common delimiters.

::component-preview{file="SAvatar/initials"}
::

## With Image

Use the `src` prop to display an image instead of initials.

::component-preview{file="SAvatar/with-image"}
::

## Borderless

Set `borderless` to `true` to remove the outline border.

::component-preview{file="SAvatar/borderless"}
::

## Indicator

Set `indicator` to `true` to show a status indicator dot. Use `indicatorPosition` to control its placement.

::component-preview{file="SAvatar/indicator"}
::

## Indicator with Image

The indicator also works when displaying an image.

::component-preview{file="SAvatar/indicator-image"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | `'?'` | Name used to generate initials. Also used as `alt` text for images |
| `src` | `string` | `''` | Image URL. When provided, displays an image instead of initials |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | Avatar size |
| `borderless` | `boolean` | `false` | Removes the outline border |
| `indicator` | `boolean` | `false` | Shows a status indicator dot |
| `indicatorPosition` | `'left-top' \| 'left-bottom' \| 'right-top' \| 'right-bottom'` | `'right-top'` | Position of the indicator dot |
| `class` | `string` | - | Additional CSS classes |
