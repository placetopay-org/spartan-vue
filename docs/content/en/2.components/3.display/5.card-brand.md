# Usage

The `SCardBrand` component renders payment method and card brand logos. It supports a wide variety of brands including Visa, Mastercard, Amex, PSE, and many more.

::component-preview{file="SCardBrand/basic"}
::

## Size

Use the `size` prop to control the width and height of the brand icon in pixels.

```vue
<template>
  <SCardBrand name="visa" :size="64" />
</template>
```

## All Brands

Below is the complete list of available brand names:

`account`, `ach`, `alkosto`, `amex`, `athCard`, `athmv`, `avalButton`, `bancolombiaButton`, `cash`, `codensa`, `comfandi`, `compensarWallet`, `credencial`, `debit`, `diners`, `discover`, `elo`, `epm`, `equity`, `exito`, `ganapin`, `jcb`, `maestro`, `master`, `mefia`, `pagoEfectivo`, `paypal`, `pse`, `ris`, `safetyPay`, `somos`, `tarjetaClave`, `transerverPoints`, `tuya`, `visaElectron`, `visa`

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `'visa' \| 'master' \| 'amex' \| ...` | - | The brand name to display. See the full list above. |
| `size` | `number` | - | Width and height of the SVG icon in pixels. |
| `class` | `string` | - | Additional CSS classes to apply to the SVG element. |
