# Usage

The SPageTitle component renders an h1 element with predefined text styles (2xl size, semibold weight, gray-900 color).

::spartan-component
---
props: {}
slots:
  default: Dashboard Overview
ignore:
  - class
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes to customize the appearance |

## Slots

| Name | Description |
|------|-------------|
| `default` | The text content of the page title |
