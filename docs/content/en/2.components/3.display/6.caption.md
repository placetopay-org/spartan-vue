# Usage

The SCaption component displays small helper or error text, commonly used below form inputs to provide additional context or validation feedback.

::component-preview{file="SCaption/basic"}
::

## Variants

SCaption supports two visual variants that determine the text color and semantic meaning.

::component-preview{file="SCaption/variants"}
::

## With Custom Content

You can use the default slot to provide custom content instead of using the `text` prop. This is useful when you need formatted text or multiple elements.

::component-preview{file="SCaption/customContent"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'error' \| 'info'` | `'error'` | The visual variant of the caption that determines the text color |
| `text` | `string` | - | The text content to display in the caption |

## Slots

| Name | Description |
|------|-------------|
| `default` | Custom content that replaces the text prop |
