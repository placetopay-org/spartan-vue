# Usage

The `SInputIncrement` component renders a numeric input with increment and decrement buttons. Use `v-model` to bind the numeric value.

::component-preview{file="SInputIncrement/basic"}
::

## Min and Max

Use the `min` and `max` props to constrain the value range. The buttons are automatically disabled when the value reaches the limits.

::component-preview{file="SInputIncrement/min-max"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInputIncrement/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SInputIncrement/disabled"}
::

## Block

The `SInputIncrementBlock` component wraps `SInputIncrement` with a label, help text, and error text. It combines all the base component props with `label`, `helpText`, and `errorText`. When `errorText` is provided, the error state is automatically applied.

::component-preview{file="SInputIncrement/block"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `number` | - | The numeric value (v-model) |
| `min` | `number` | - | Minimum allowed value |
| `max` | `number` | - | Maximum allowed value |
| `error` | `boolean` | `false` | Displays error styling |
| `disabled` | `boolean` | `false` | Disables the input and buttons |
| `containerClass` | `string` | - | Additional CSS classes for the container |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `number` | Emitted when the numeric value changes |
