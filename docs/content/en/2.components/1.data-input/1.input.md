# Usage

The `SInput` component renders a text input field. Use `v-model` to bind the input value.

::component-preview{file="SInput/basic"}
::

## Prefix and Suffix

Use the `prefix` and `suffix` props to display static text before or after the input.

::component-preview{file="SInput/prefix-suffix"}
::

## Rounded

Use the `rounded` prop to control the border radius of the input.

::component-preview{file="SInput/rounded"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInput/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SInput/disabled"}
::

## Borderless

Set `borderless` to `true` to remove the input border.

::component-preview{file="SInput/borderless"}
::

## Left Icon

Use the `leftIcon` prop to display an icon on the left side of the input.

::component-preview{file="SInput/left-icon"}
::

## Right Icon

Use the `rightIcon` prop to display an icon on the right side of the input.

::component-preview{file="SInput/right-icon"}
::

## Selectors

Use `leftOptions` or `rightOptions` to embed a dropdown selector inside the input. Bind the selected value with `v-model:leftOption` or `v-model:rightOption`.

::component-preview{file="SInput/selectors"}
::

## Combined

Combine multiple features like icons, prefix, and error state for real-world use cases.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string \| number \| null` | - | The input value (v-model) |
| `type` | `string` | `'text'` | The input type attribute (text, email, number, etc.) |
| `placeholder` | `string` | - | Placeholder text |
| `name` | `string` | - | The input name attribute |
| `id` | `string` | - | The input id attribute |
| `disabled` | `boolean` | `false` | Disables the input |
| `error` | `boolean` | `false` | Displays error styling |
| `borderless` | `boolean` | `false` | Removes the border |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |
| `prefix` | `string` | - | Static text displayed before the input |
| `suffix` | `string` | - | Static text displayed after the input |
| `leftIcon` | `FunctionalComponent` | - | Icon displayed on the left side |
| `rightIcon` | `FunctionalComponent` | - | Icon displayed on the right side |
| `leftOptions` | `TOption[]` | - | Dropdown selector options on the left |
| `rightOptions` | `TOption[]` | - | Dropdown selector options on the right |
| `leftOption` | `string` | - | Selected value for left selector (v-model:leftOption) |
| `rightOption` | `string` | - | Selected value for right selector (v-model:rightOption) |
| `leftOrderSlots` | `string` | `'selector,text,icon,slot'` | Render order for left side sections |
| `rightOrderSlots` | `string` | `'slot,icon,text,selector'` | Render order for right side sections |
| `class` | `string` | - | Additional CSS classes for the container |
| `inputClass` | `string` | - | Additional CSS classes for the input element |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string \| number \| undefined` | Emitted when the input value changes |
| `update:leftOption` | `string \| undefined` | Emitted when the left selector value changes |
| `update:rightOption` | `string \| undefined` | Emitted when the right selector value changes |

## Slots

| Name | Description |
|------|-------------|
| `left` | Custom content on the left side of the input |
| `right` | Custom content on the right side of the input |
