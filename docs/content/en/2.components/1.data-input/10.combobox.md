# Usage

The `SCombobox` component renders an accessible dropdown selector. Use `SComboboxOption` children to define the available options and `displayButtonText` to control the displayed text.

::component-preview{file="SCombobox/basic"}
::

## Search

Set `search` to `'auto'` to enable a built-in search input that filters options automatically. The `displayButtonText` function is required when search is enabled.

::component-preview{file="SCombobox/search"}
::

## Option Groups

Use `SComboboxOptionGroup` to organize options into labeled groups.

::component-preview{file="SCombobox/groups"}
::

## Error

Set `error` to `true` to display the combobox with an error style.

::component-preview{file="SCombobox/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SCombobox/disabled"}
::

## Props

### SCombobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string \| number \| object` | - | The selected value (v-model) |
| `search` | `true \| 'auto'` | - | Enables search input (`'auto'` filters options locally) |
| `displayButtonText` | `(option: unknown) => string` | - | Function to display the selected value text |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |
| `error` | `boolean` | `false` | Displays error styling |
| `disabled` | `boolean` | `false` | Disables the combobox |
| `flipOptions` | `boolean` | `false` | Aligns dropdown to the right |
| `queryDebounce` | `number` | `500` | Debounce time in ms for search queries |
| `id` | `string` | - | HTML id attribute |
| `class` | `string` | - | Additional CSS classes |

### SComboboxOption

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string \| number \| object` | - | The option value |
| `disabled` | `boolean` | `false` | Disables the option |
| `class` | `string` | - | Additional CSS classes |

### SComboboxOptionGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | The group label text |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string \| number \| object` | Emitted when the selected value changes |
| `query` | `string` | Emitted when the search query changes |

## Slots

### SCombobox

| Name | Description |
|------|-------------|
| `default` | `SComboboxOption` or `SComboboxOptionGroup` components |
| `button` | Custom content for the button display |

### SComboboxOption

| Name | Description |
|------|-------------|
| `default` | The option text content |

### SComboboxOptionGroup

| Name | Description |
|------|-------------|
| `default` | `SComboboxOption` components within this group |
