# Usage

The `SInputOtp` component renders a numeric OTP input. Place `SInputOtpItem` children inside to define the number of digits.

::component-preview{file="SInputOtp/basic"}
::

## Six Digits

Add more `SInputOtpItem` children to increase the number of digits.

::component-preview{file="SInputOtp/six-digits"}
::

## Success

Set `success` to `true` to display the input with a success style.

::component-preview{file="SInputOtp/success"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInputOtp/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SInputOtp/disabled"}

## Props

### SInputOtp

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The OTP value (v-model) |
| `disabled` | `boolean` | `false` | Disables the input |
| `success` | `boolean` | `false` | Displays success styling |
| `error` | `boolean` | `false` | Displays error styling |
| `class` | `string` | - | Additional CSS classes |

### SInputOtpItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes for the digit slot |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the OTP value changes |

## Slots

| Name | Description |
|------|-------------|
| `default` | `SInputOtpItem` components defining the digit slots |
