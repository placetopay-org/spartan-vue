# Usage

The `SInputPassword` component wraps `SInput` with a visibility toggle button. Click the eye icon to show or hide the password.

::component-preview{file="SInputPassword/basic"}
::

## Meter

Set `meter` to `true` to display a color bar at the bottom of the input indicating password validity. Combine with `rules` for full validation.

::component-preview{file="SInputPassword/meter"}
::

## Validation Rules

Use the `rules` prop to define password validation criteria. Combine with the `SInputPasswordPanel` component to display real-time validation feedback.

::component-preview{file="SInputPassword/rules"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SInputPassword/disabled"}
::

## Props

### SInputPassword

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string \| number \| null` | - | The password value (v-model) |
| `rules` | `TRules` | - | Validation rules object |
| `meter` | `boolean` | `false` | Shows the strength meter bar |
| `placeholder` | `string` | - | Placeholder text |
| `disabled` | `boolean` | `false` | Disables the input |
| `error` | `boolean` | `false` | Displays error styling |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |

The component also accepts all `SInput` props.

#### Rules Object

| Rule | Type | Description |
|------|------|-------------|
| `min` | `number` | Minimum character length |
| `max` | `number` | Maximum character length |
| `lowercase` | `boolean` | Requires at least one lowercase letter |
| `uppercase` | `boolean` | Requires at least one uppercase letter |
| `digit` | `boolean` | Requires at least one digit |
| `special` | `boolean` | Requires at least one special character |

### SInputPasswordPanel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `state` | `TState` | - | The validation state from the `state` event |
| `open` | `boolean` | `true` | Whether the panel is expanded |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the password value changes |
| `state` | `TState` | Emitted with the validation state object |
| `isValid` | `boolean` | Emitted when the overall validity changes |
