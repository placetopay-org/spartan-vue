# Usage

The `SInputTag` component renders a tag input field. Use `v-model` to bind an array of strings. Press **Enter** to add a tag.

::component-preview{file="SInputTag/basic"}
::

## Rounded

Use the `rounded` prop to control the border radius of the container.

::component-preview{file="SInputTag/rounded"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInputTag/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SInputTag/disabled"}
::

## Borderless

Set `borderless` to `true` to remove the container border.

::component-preview{file="SInputTag/borderless"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string[]` | `[]` | The array of tags (v-model) |
| `placeholder` | `string` | - | Placeholder text for the input |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |
| `error` | `boolean` | `false` | Displays error styling |
| `disabled` | `boolean` | `false` | Disables the input |
| `borderless` | `boolean` | `false` | Removes the border |
| `type` | `string` | `'text'` | The input type attribute |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string[]` | Emitted when the tags array changes |
