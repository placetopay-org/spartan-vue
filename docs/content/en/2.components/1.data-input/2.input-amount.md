# Usage

The `SInputAmount` component renders a formatted numeric input for currency amounts. It automatically formats the value based on the selected currency's decimal precision and locale. Use `v-model` to bind the numeric value and `currency` to set the active currency.

::spartan-component
---
props:
  currency: USD
  symbol: true
  suffixCurrency: false
  disabled: false
  error: false
  placeholder: Enter amount
model:
  - modelValue
ignore:
  - class
  - inputClass
  - leftIcon
  - rightIcon
  - leftOption
  - leftOptions
  - rightOption
  - rightOptions
  - leftOrderSlots
  - rightOrderSlots
  - prefix
  - suffix
  - rounded
  - type
  - borderless
  - id
  - name
  - currencies
  - locale
  - minorUnitMode
---
::

## Minor Unit Mode

Set `minorUnitMode` to `true` to work with minor units (e.g., cents). The `modelValue` will be emitted as the amount multiplied by 10^decimals. For example, `$22.99` USD becomes `2299`.

::spartan-component
---
props:
  currency: USD
  minorUnitMode: true
  symbol: true
  placeholder: 0.00
model:
  - modelValue
ignore:

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `number \| null` | - | The numeric amount value (v-model) |
| `currency` | `string` | - | The active ISO 4217 currency code (e.g., `'USD'`, `'EUR'`, `'COP'`) |
| `currencies` | `string[]` | - | Array of currency codes to display a currency selector dropdown |
| `symbol` | `boolean` | `false` | Display the currency symbol as a prefix |
| `suffixCurrency` | `boolean` | `false` | Display the currency code as a suffix |
| `locale` | `string` | - | BCP 47 locale string for number formatting (e.g., `'en-US'`, `'es-CO'`) |
| `minorUnitMode` | `boolean` | `false` | Emit and accept values in minor units (cents) |
| `placeholder` | `string` | - | Placeholder text |
| `disabled` | `boolean` | `false` | Disables the input |
| `error` | `boolean` | `false` | Displays error styling |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |
| `borderless` | `boolean` | `false` | Removes the border |
| `class` | `string` | - | Additional CSS classes for the container |
| `inputClass` | `string` | - | Additional CSS classes for the input element |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `number \| null` | Emitted when the amount value changes |
| `update:currency` | `string \| undefined` | Emitted when the selected currency changes |
| `info` | `AmountInfo` | Emitted with detailed amount info including currency, decimals, code, symbol, and minor unit |
| `change` | `number \| null` | Emitted on value change (required by vue-currency-input) |

## Slots

| Name | Description |
|------|-------------|
| `left` | Custom content on the left side of the input |
| `right` | Custom content on the right side of the input |
