# Usage

The `SSelect` component renders a native HTML `<select>` element. Use `v-model` to bind the selected value and provide `<option>` elements via the default slot.

::component-preview{file="SSelect/basic"}
::

## Placeholder

Use the `placeholder` prop to display a disabled placeholder option when no value is selected.

::component-preview{file="SSelect/placeholder"}
::

## Rounded

Use the `rounded` prop to control the border radius of the select.

::component-preview{file="SSelect/rounded"}
::

## Error

Set `error` to `true` to display the select with an error style.

::component-preview{file="SSelect/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="SSelect/disabled"}
::

## Block

The `SSelectBlock` component wraps `SSelect` with a label, help text, and error text. It combines all the base component props with `label`, `helpText`, and `errorText`. When `errorText` is provided, the error state is automatically applied.

::component-preview{file="SSelect/block"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string \| number` | - | The selected value (v-model) |
| `placeholder` | `string` | - | Disabled placeholder option text |
| `rounded` | `'both' \| 'left' \| 'right' \| 'none' \| 'full'` | `'both'` | Border radius style |
| `error` | `boolean` | `false` | Displays error styling |
| `disabled` | `boolean` | `false` | Disables the select |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string \| number \| undefined` | Emitted when the selected value changes |

## Slots

| Name | Description |
|------|-------------|
| `default` | The `<option>` elements to render inside the select |
