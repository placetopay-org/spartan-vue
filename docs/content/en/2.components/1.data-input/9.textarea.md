# Usage

The `STextArea` component renders a native `<textarea>` element. Use `v-model` to bind the text value.

::component-preview{file="STextarea/basic"}
::

## Error

Set `error` to `true` to display the textarea with an error style.

::component-preview{file="STextarea/error"}
::

## Disabled

Set `disabled` to `true` to prevent user interaction.

::component-preview{file="STextarea/disabled"}
::

## Block

The `STextAreaBlock` component wraps `STextArea` with a label, help text, and error text. It combines all the base component props with `label`, `helpText`, and `errorText`. When `errorText` is provided, the error state is automatically applied.

::component-preview{file="STextarea/block"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The textarea value (v-model) |
| `error` | `boolean` | `false` | Displays error styling |
| `disabled` | `boolean` | `false` | Disables the textarea |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the textarea value changes |
