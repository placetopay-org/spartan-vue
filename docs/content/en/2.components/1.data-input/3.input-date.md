# Usage

The `SInputDate` component renders a date picker input. Use `v-model` to bind the selected date value. The model value is a native `Date` object.

::component-preview{file="SInputDate/basic"}
::

## Format

Use the `dateFormat` prop to control how the date is displayed in the input (e.g. `'dd/mm/yy'`, `'mm/dd/yy'`, `'yy-mm-dd'`).

::component-preview{file="SInputDate/format"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInputDate/error"}
::

## Show Icon

Set `showIcon` to `false` to hide the calendar button. By default the icon is shown.

::component-preview{file="SInputDate/show-icon"}
::

## Block

The `SInputDateBlock` component wraps `SInputDate` with a label, help text, and error text. It combines all the base component props with `label`, `helpText`, and `errorText`. When `errorText` is provided, the error state is automatically applied.

::component-preview{file="SInputDate/block"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `Date \| Date[] \| null` | - | The selected date value (v-model) |
| `placeholder` | `string` | - | Placeholder text for the input |
| `error` | `boolean` | `false` | Displays error styling |
| `showIcon` | `boolean` | `true` | Shows the calendar button |
| `id` | `string` | - | HTML id attribute for the input |
| `dateFormat` | `string` | - | Date display format (e.g. `'dd/mm/yy'`) |
| `selectionMode` | `'single' \| 'multiple' \| 'range'` | `'single'` | Selection behavior |
| `showTime` | `boolean` | `false` | Displays time picker alongside the calendar |
| `hourFormat` | `'12' \| '24'` | - | 12 or 24 hour time format |
| `showButtonBar` | `boolean` | `false` | Displays today and clear buttons |
| `minDate` | `Date` | - | Minimum selectable date |
| `maxDate` | `Date` | - | Maximum selectable date |
| `disabled` | `boolean` | `false` | Disables the component |
| `readonly` | `boolean` | `false` | Makes the component read-only |
| `manualInput` | `boolean` | - | Allows typing a date into the input |
| `disabledDates` | `Date[]` | - | Array of dates to disable |
| `numberOfMonths` | `number` | `1` | Number of months to display |
| `showWeek` | `boolean` | `false` | Shows week numbers |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `Date \| Date[] \| null` | Emitted when the selected date changes |
