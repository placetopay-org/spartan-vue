# Usage

The `SInputDate` component renders a date picker input. Use `v-model` to bind the selected date value.

::component-preview{file="SInputDate/basic"}
::

## Format

Use the `modelType` prop to control the date format string used for the model value.

::component-preview{file="SInputDate/format"}
::

## Error

Set `error` to `true` to display the input with an error style.

::component-preview{file="SInputDate/error"}
::

## Hide Icon

Set `hideInputIcon` to `true` to hide the calendar icon inside the input.

::component-preview{file="SInputDate/hide-icon"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string \| string[] \| null` | - | The selected date value (v-model) |
| `placeholder` | `string` | - | Placeholder text for the input |
| `error` | `boolean` | `false` | Displays error styling |
| `hideInputIcon` | `boolean` | `false` | Hides the calendar icon |
| `id` | `string` | - | HTML id attribute for the input |
| `modelType` | `string` | - | Date format string (e.g. `'dd-MM-yyyy'`) |
| `format` | `string` | - | Display format for the date |
| `class` | `string` | - | Additional CSS classes |

The component also accepts all [VueDatePicker](https://vue3datepicker.com/) props.

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string \| string[] \| null` | Emitted when the selected date changes |
