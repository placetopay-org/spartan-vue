# Usage

The `SSectionTitle` component renders a styled heading element with dark mode support. Use the `as` prop to control which HTML heading tag is used.

::component-preview{file="SSectionTitle/basic"}
::

## Heading levels

Use the `as` prop to render different HTML heading elements (`h1` through `h6`).

::component-preview{file="SSectionTitle/heading-levels"}
::

## Custom styles

Pass a `class` prop to override or extend the default styles. Classes are merged intelligently using `tailwind-merge`.

::component-preview{file="SSectionTitle/custom-styles"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6'` | `'h3'` | The HTML heading element to render |
| `class` | `string` | - | Additional CSS classes to customize appearance |

## Slots

| Name | Description |
|------|-------------|
| `default` | Heading text content |
