# Usage

The `SSectionDescription` component renders styled descriptive text with dark mode support. Use the `as` prop to control which HTML element is used.

::component-preview{file="SSectionDescription/basic"}
::

## HTML element

Use the `as` prop to render a different HTML element. Defaults to `p`.

::component-preview{file="SSectionDescription/element"}
::

## Custom styles

Pass a `class` prop to override or extend the default styles. Classes are merged intelligently using `tailwind-merge`.

::component-preview{file="SSectionDescription/custom-styles"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `string` | `'p'` | The HTML element to render |
| `class` | `string` | - | Additional CSS classes to customize appearance |

## Slots

| Name | Description |
|------|-------------|
| `default` | Description text content |
