# Usage

The SPageTitle component renders a semantic `h1` element with predefined text styles (2xl size, semibold weight) and dark mode support. Use it to display the main heading of any page with consistent typography.

::component-preview{file="SPageTitle/basic"}
::

## Custom class

You can override the default styles using the `class` prop. The component uses `tailwind-merge` to intelligently merge your custom classes with the defaults.

::component-preview{file="SPageTitle/custom-class"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `HTMLAttributes['class']` | - | Additional CSS classes to customize the appearance |

## Slots

| Name | Description |
|------|-------------|
| `default` | The text content of the page title |
