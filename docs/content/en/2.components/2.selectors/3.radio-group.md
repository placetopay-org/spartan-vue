# Usage

The `SRadioGroup` component wraps HeadlessUI's `RadioGroup` to provide a card-based radio selection interface. Use `SRadioGroupItem` inside to define each option with a title, description, and footer.

::component-preview{file="SRadioGroup/basic"}
::

## Simple

Use `SRadioGroupItem` with just a title slot for minimal selection cards without descriptions or footers.

::component-preview{file="SRadioGroup/simple"}
::

## Disabled

Use the `disabled` prop on `SRadioGroup` to disable all options at once, preventing any user interaction.

::component-preview{file="SRadioGroup/disabled"}
::

## Disabled Item

Use the `disabled` prop on individual `SRadioGroupItem` components to disable specific options while keeping the rest interactive.

::component-preview{file="SRadioGroup/disabled-item"}
::

## SRadioGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The currently selected value. Required. |
| `disabled` | `boolean` | `false` | Disables all radio group items, preventing user interaction. |

## SRadioGroup Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the selected option changes. |

## SRadioGroup Slots

| Name | Description |
|------|-------------|
| `default` | Container for `SRadioGroupItem` components. |

## SRadioGroupItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The value this option represents. Required. Must be unique within the group. |
| `disabled` | `boolean` | `false` | Disables this specific option, preventing user interaction. |

## SRadioGroupItem Slots

| Name | Description |
|------|-------------|
| `title` | The main label for the option. |
| `description` | Additional description text below the title. |
| `footer` | Footer content, typically used for pricing or metadata. |
