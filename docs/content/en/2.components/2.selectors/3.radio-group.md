# Usage

The `SRadioGroup` component wraps HeadlessUI's `RadioGroup` to provide a card-based radio selection interface. Use `SRadioGroupItem` inside to define each option with a title, description, and footer.

::spartan-component
---
props:
  modelValue: startup
model:
  - modelValue
slots:
  default:
    children:
      - component: SRadioGroupItem
        props:
          value: startup
        slots:
          title: Startup
          description: 12GB / 6 CPUs
          footer: $40/mo
      - component: SRadioGroupItem
        props:
          value: business
        slots:
          title: Business
          description: 16GB / 8 CPUs
          footer: $80/mo
      - component: SRadioGroupItem
        props:
          value: enterprise
        slots:
          title: Enterprise
          description: 32GB / 12 CPUs
          footer: $160/mo
---
::

## Simple Usage

You can use `SRadioGroupItem` with just a title slot for simpler selection cards.

::spartan-component
---
props:
  modelValue: free
model:
  - modelValue
slots:
  default:
    children:
      - component: SRadioGroupItem
        props:
          value: free
        slots:
          title: Free
      - component: SRadioGroupItem
        props:
          value: pro
        slots:
          title: Pro
---
::

## SRadioGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `string` | - | The currently selected value. Required. |

## SRadioGroup Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `string` | Emitted when the selected option changes. |

## SRadioGroup Slots

| Name | Description |
|------|-------------|
| `default` | Container for `SRadioGroupItem` components. |

## SRadioGroupItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The value this option represents. Required. |

## SRadioGroupItem Slots

| Name | Description |
|------|-------------|
| `title` | The main label for the option. |
| `description` | Additional description text below the title. |
| `footer` | Footer content, typically used for pricing or metadata. |
