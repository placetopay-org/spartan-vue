# Usage

The `SRadio` component provides a styled radio input with optional label and description slots. Use the `name` prop to group radios together and `value` to identify each option.

::spartan-component
---
props:
  value: option1
  name: example
  disabled: false
  reverse: false
  inline: false
slots:
  default: Option 1
  description: Select this option to continue.
ignore:
  - id
  - modelValue
  - class
---
::

## Disabled

Use the `disabled` prop to prevent user interaction with the radio. The component will appear with reduced opacity.

::spartan-component
---
props:
  value: disabled-option
  name: disabled-example
  disabled: true
slots:
  default: Disabled option
ignore:
  - id
  - modelValue
  - reverse
  - inline
  - class
---
::

## Reverse

Use the `reverse` prop to place the radio on the right side with the label on the left.

::spartan-component
---
props:
  value: reversed
  name: reverse-example
  reverse: true
slots:
  default: Reversed radio
  description: The radio appears on the right side.
ignore:
  - id
  - modelValue
  - disabled
  - inline
  - class
---
::

## Inline

Use the `inline` prop to render the label inline with the description text instead of above it.

::spartan-component
---
props:
  value: inline-option
  name: inline-example
  inline: true
slots:
  default: Inline radio
  description: The label appears inline with the description.
ignore:
  - id
  - modelValue
  - disabled
  - reverse
  - class
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `boolean \| string` | - | The bound value for the radio group. Required. |
| `value` | `string` | - | The value this radio represents. Required. |
| `disabled` | `boolean` | `false` | Disables the radio and reduces opacity. |
| `reverse` | `boolean` | `false` | Places the radio on the right side of the label. |
| `inline` | `boolean` | `false` | Renders the label inline with the description. |
| `name` | `string` | - | The HTML `name` attribute to group radios together. |
| `id` | `string` | auto-generated | The HTML `id` attribute. Auto-generated if not provided. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `boolean \| string` | Emitted when the radio is selected. |

## Slots

| Name | Description |
|------|-------------|
| `default` | The label text for the radio. |
| `description` | Additional description text displayed below or beside the label. |
