# Usage

The `SSwitch` component provides a styled toggle switch built on HeadlessUI. It supports labels, descriptions, icons, and passive mode.

::component-preview{file="SSwitch/basic"}
::

## Reverse

Use the `reverse` prop to place the switch on the right side with the label on the left.

::component-preview{file="SSwitch/reverse"}
::

## Passive

When `passive` is enabled, clicking the label or description does not toggle the switch. Only clicking the switch itself changes the value.

::component-preview{file="SSwitch/passive"}
::

## With Icon

Set the `icon` prop to `true` to show default check/cross icons inside the switch knob.

::component-preview{file="SSwitch/with-icon"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `boolean` | - | The current toggle state. Required. |
| `reverse` | `boolean` | `false` | Places the switch on the right side of the label. |
| `passive` | `boolean` | `false` | Prevents label/description clicks from toggling the switch. |
| `icon` | `boolean \| FunctionalComponent` | - | Shows default icons when `true`, or a custom icon component for both states. |
| `iconOn` | `FunctionalComponent` | - | Custom icon component for the ON state. |
| `iconOff` | `FunctionalComponent` | - | Custom icon component for the OFF state. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `boolean` | Emitted when the switch is toggled. |

## Slots

| Name | Description |
|------|-------------|
| `default` | The label text for the switch. |
| `description` | Additional description text below the label. |
