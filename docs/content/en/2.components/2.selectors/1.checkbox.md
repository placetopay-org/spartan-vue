# Usage

The `SCheckbox` component provides a styled checkbox input with optional label and description slots. It supports both single boolean and array-based group selection via `v-model`.

::component-preview{file="SCheckbox/basic"}
::

## Disabled

Use the `disabled` prop to prevent user interaction with the checkbox. The component will appear with reduced opacity.

::component-preview{file="SCheckbox/disabled"}
::

## Reverse

Use the `reverse` prop to place the checkbox on the right side with the label on the left, using justify-between layout.

::component-preview{file="SCheckbox/reverse"}
::

## Inline

Use the `inline` prop to render the label inline with the description text instead of above it.

::component-preview{file="SCheckbox/inline"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `boolean \| string[]` | - | The bound value. Use `boolean` for single checkboxes or `string[]` for groups. |
| `value` | `string` | - | The value identifier when used in a group with an array `modelValue`. |
| `disabled` | `boolean` | `false` | Disables the checkbox and reduces opacity. |
| `reverse` | `boolean` | `false` | Places the checkbox on the right side of the label. |
| `inline` | `boolean` | `false` | Renders the label inline with the description. |
| `name` | `string` | - | The HTML `name` attribute for the input. |
| `id` | `string` | auto-generated | The HTML `id` attribute. Auto-generated if not provided. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `boolean \| string[]` | Emitted when the checkbox state changes. |

## Slots

| Name | Description |
|------|-------------|
| `default` | The label text for the checkbox. |
| `description` | Additional description text displayed below or beside the label. |
