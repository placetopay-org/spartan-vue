# Usage

The `SLabel` component is a styled label element for form inputs, providing consistent typography and accessibility options. Use the `for` prop to associate it with a form input by its ID.

::component-preview{file="SLabel/basic"}
::

## Screen reader only

Use the `srOnly` prop to visually hide the label while keeping it accessible to screen readers.

::component-preview{file="SLabel/sr-only"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `for` | `string` | - | The ID of the input element this label is associated with |
| `srOnly` | `boolean` | `false` | Visually hide the label (screen reader only) |

## Slots

| Name | Description |
|------|-------------|
| `default` | Label text content |
