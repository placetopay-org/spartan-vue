# Usage

The `SCopy` component allows users to copy text to the clipboard with a single click. It displays a copy icon that transitions to a success state and shows a tooltip when the text is copied.

::spartan-component
---
props:
  value: 'Hello World'
slots:
  default: Hello World
---
::

## With value prop

Use the `value` prop to specify the text to copy. If not provided, the component will copy the text content of the default slot.

::spartan-component
---
props:
  value: 'Custom text to copy'
slots:
  default: Click to copy custom text
---
::

## Without value prop

When no `value` prop is provided, the component automatically copies the text content rendered inside the default slot.

::spartan-component
---
slots:
  default: This text will be copied
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Text to copy to the clipboard. If not provided, copies the slot content |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `copied` | `string` | Emitted when text is copied, with the copied value |

## Slots

| Name | Description |
|------|-------------|
| `default` | Content displayed next to the copy icon. Also used as fallback text to copy if `value` is not provided |
