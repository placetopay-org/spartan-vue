## Usage

The `SColorSwitch` component provides a compact toggle to switch between system, light, and dark color modes. The active mode shows a bold icon with a highlighted background.

::component-preview{file="SColorSwitch/basic"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modelValue` | `'system' \| 'light' \| 'dark'` | `'system'` | Current color mode |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `TColorSwitchMode` | Emitted when the color mode changes |
