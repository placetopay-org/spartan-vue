# Usage

The `SButton` component is a versatile button that supports multiple visual variants, sizes, icons, and states. The `variant` prop defines the color palette, while `outline`, `link`, and `circular` are independent boolean modifiers.

::component-preview{file="SButton/basic"}
::

## Variants

Use the `variant` prop to change the button's color palette.

::component-preview{file="SButton/variants"}
::

## Outline

Use the `outline` prop to render the button with a transparent background, keeping the border and text in the variant color.

::component-preview{file="SButton/outline"}
::

## Link

Use the `link` prop to render the button without background or border, keeping the text in the variant color with hover effect.

::tip
If both `link` and `outline` are set, `link` takes priority.
::

::component-preview{file="SButton/link"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'danger'` | `'primary'` | Color palette of the button |
| `outline` | `boolean` | `false` | Transparent background with border and text in variant color |
| `link` | `boolean` | `false` | No background or border, text in variant color |
| `circular` | `boolean` | `false` | Round icon-only button |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `disabled` | `boolean` | `false` | Disables the button |
| `loading` | `boolean` | `false` | Shows loading state |
| `icon` | `FunctionalComponent` | - | Shortcut for `leftIcon` |
| `leftIcon` | `FunctionalComponent` | - | Icon on the left side |
| `rightIcon` | `FunctionalComponent` | - | Icon on the right side |
| `rounded` | `'none' \| 'left' \| 'right' \| 'both' \| 'full'` | `'both'` | Border radius |
| `type` | `'button' \| 'submit'` | `'button'` | HTML button type |
| `as` | `string \| Component` | `'button'` | Element to render |
