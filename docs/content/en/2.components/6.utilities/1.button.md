# Usage

The `SButton` component is a versatile button that supports multiple visual variants, sizes, icons, and states.

::spartan-component
---
props:
  variant: primary
  size: md
  disabled: false
  loading: false
slots:
  default: Click me
---
::

## Variants

Use the `variant` prop to change the button's visual style.

::spartan-component
---
items: { variant: ['primary', 'secondary', 'danger', 'outline', 'link'] }
props:
  variant: primary
slots:
  default: Variants
---
::

## Sizes

Use the `size` prop to change the button size.

::spartan-component
---
items: { size: ['sm', 'md', 'lg'] }
props:
  size: md
slots:
  default: Sizes
---
::

## States

### Disabled

::spartan-component
---
props:
  disabled: true
slots:
  default: Disabled
---
::

### Loading

::spartan-component
---
props:
  loading: true
  variant: primary
slots:
  default: Loading...
---
::

## Rounded

Use the `rounded` prop to control the border radius.

::spartan-component
---
items: { rounded: ['none', 'left', 'right', 'both', 'full'] }
props:
  rounded: both
slots:
  default: Rounded
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'danger' \| 'outline' \| 'link'` | `'primary'` | Visual variant of the button |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `disabled` | `boolean` | `false` | Disables the button |
| `loading` | `boolean` | `false` | Shows loading state |
| `leftIcon` | `FunctionalComponent` | - | Icon on the left side |
| `rightIcon` | `FunctionalComponent` | - | Icon on the right side |
| `rounded` | `'none' \| 'left' \| 'right' \| 'both' \| 'full'` | `'both'` | Border radius |
| `type` | `'button' \| 'submit'` | `'button'` | HTML button type |
| `as` | `string \| Component` | `'button'` | Element to render |

## Slots

| Name | Description |
|------|-------------|
| `default` | Button content |
