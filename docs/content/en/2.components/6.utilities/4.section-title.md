# Usage

The `SSectionTitle` component renders a styled heading element. Use the `as` prop to control which HTML heading tag is used.

::spartan-component
---
props:
  as: h3
slots:
  default: Section Title
---
::

## Heading levels

Use the `as` prop to render different HTML heading elements (`h1` through `h6`).

::spartan-component
---
items: { as: ['h1', 'h2', 'h3', 'h4', 'h5', 'h6'] }
props:
  as: h3
slots:
  default: Heading Level
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6'` | `'h3'` | The HTML heading element to render |
| `class` | `string` | - | Additional CSS classes |

## Slots

| Name | Description |
|------|-------------|
| `default` | Heading text content |
