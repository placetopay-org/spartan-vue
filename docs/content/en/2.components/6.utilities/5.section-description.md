# Usage

The `SSectionDescription` component renders styled descriptive text. Use the `as` prop to control which HTML element is used.

::spartan-component
---
props:
  as: p
slots:
  default: This is a section description that provides additional context about the content.
---
::

## HTML element

Use the `as` prop to render a different HTML element. Defaults to `p`.

::spartan-component
---
items: { as: ['p', 'span', 'div'] }
props:
  as: p
slots:
  default: Description rendered with a different element.
---
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `string` | `'p'` | The HTML element to render |
| `class` | `string` | - | Additional CSS classes |

## Slots

| Name | Description |
|------|-------------|
| `default` | Description text content |
