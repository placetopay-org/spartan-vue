# Usage

`SDropdown` provides a click-triggered menu built on top of `SPopover` and HeadlessUI's `Menu` component. Use `SDropdownItem` for each menu entry.

The `#reference` slot defines the trigger button content. The default slot accepts `SDropdownItem` components.

::

## Placement

Use the `placement` prop to control where the dropdown menu appears relative to the trigger.

## With Icons

`SDropdownItem` supports an `icon` prop that accepts a `FunctionalComponent`. Use the `#description` slot to add secondary text below the item label.

## Disabled Items

Set the `disabled` prop on `SDropdownItem` to prevent interaction with specific menu items.

## SDropdown Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placement` | `Placement` | `'bottom-start'` | Position of the dropdown menu relative to the trigger |
| `arrow` | `'auto' \| 'light' \| 'dark'` | - | Displays an arrow pointing to the trigger |
| `offset` | `number` | `0` | Distance in pixels between the trigger and the menu |
| `static` | `boolean` | `false` | Disables automatic flip when the menu would overflow the viewport |
| `responsive` | `boolean` | `true` | On mobile, displays as a bottom sheet instead of a floating panel |
| `preventClose` | `boolean` | `false` | Prevents the dropdown from closing when focus leaves |
| `preventFocus` | `boolean` | `false` | Prevents automatic focus on the menu when opened |
| `referenceClass` | `string` | - | Additional CSS classes for the trigger button |
| `floatingClass` | `string` | - | Additional CSS classes for the floating menu container |
| `manual` | `boolean` | - | Disables automatic open/close on click. Use exposed methods for manual control |
| `class` | `string` | - | Additional CSS classes for the root element |

## SDropdownItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `FunctionalComponent` | - | Icon component displayed before the label |
| `disabled` | `boolean` | - | Disables the menu item |
| `link` | `string` | - | Renders the item as an anchor (`<a>`) with this href |
| `class` | `string` | - | Additional CSS classes for the item |
| `labelClass` | `string` | - | Additional CSS classes for the label text |
| `iconClass` | `string` | - | Additional CSS classes for the icon |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `close` | - | Emitted when the dropdown closes |

## Slots

### SDropdown

| Name | Description |
|------|-------------|
| `reference` | Trigger button content |
| `default` | Menu items (`SDropdownItem` components) |

### SDropdownItem

| Name | Description |
|------|-------------|
| `default` | Item label text |
| `description` | Secondary description text below the label |
