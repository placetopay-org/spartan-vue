# Usage

`STooltip` wraps any element and displays a tooltip with the specified `title` text on hover. It is built on top of `SPopover` and supports placement, color themes, and arrow display.

::component-preview{file="STooltip/basic"}
::

## Color

Use the `color` prop to change the tooltip theme. Available values are `'auto'` (adapts to dark mode), `'dark'`, and `'light'`.

::component-preview{file="STooltip/color"}
::

## Placement

Use the `placement` prop to control where the tooltip appears relative to the wrapped element.

::component-preview{file="STooltip/placement"}
::

## Offset

Use the `offset` prop to control the distance in pixels between the trigger element and the tooltip.

::component-preview{file="STooltip/offset"}
::

## Manual Control

Set `manual` to `true` to disable automatic show/hide on hover. Then use the exposed methods (`open()`, `close()`, `toggle()`) via a template ref to control the tooltip programmatically.

::component-preview{file="STooltip/manual"}
::

## Exposed Methods

When using a template ref on `STooltip`, the following methods are available:

| Method | Description |
|--------|-------------|
| `open()` | Opens the tooltip |
| `close()` | Closes the tooltip |
| `toggle()` | Toggles the tooltip open/closed state |
| `focus()` | Focuses the underlying popover element |
| `isOpen` | Reactive ref indicating whether the tooltip is currently open |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Text content displayed in the tooltip |
| `color` | `'dark' \| 'light' \| 'auto'` | - | Color theme of the tooltip |
| `placement` | `Placement` | `'bottom'` | Position of the tooltip relative to the wrapped element |
| `arrow` | `boolean` | `true` | Whether to display an arrow pointing to the element |
| `offset` | `number` | - | Distance in pixels between the element and tooltip |
| `static` | `boolean` | - | Disables automatic flip when the tooltip would overflow the viewport |
| `manual` | `boolean` | - | Disables automatic show/hide on hover. Use exposed `open()`/`close()` methods for manual control |

## Slots

| Name | Description |
|------|-------------|
| `default` | The element that triggers the tooltip on hover |
