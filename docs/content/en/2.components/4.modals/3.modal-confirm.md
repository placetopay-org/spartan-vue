# Usage

`SModalConfirm` wraps `SModalCard` to provide a ready-to-use confirmation dialog. It includes confirm and cancel buttons out of the box, a danger icon by default, and emits a `confirm` event when the user accepts.

Use `v-model:open` to control visibility and listen to `@confirm` for the confirmation action.

::component-preview{file="SModalConfirm/basic"}
::

## With Icon

Use the `icon` prop with a preset string (`'primary'`, `'success'`, `'danger'`, `'warning'`, `'info'`) to display a predefined icon with matching colors. The default icon is `'danger'`.

::component-preview{file="SModalConfirm/with-icon"}
::

## Closable

Set `closable` to `true` to display a close button in the header. When enabled, the title and description are left-aligned.

::component-preview{file="SModalConfirm/closable"}
::

## Prevent Close

When `preventClose` is enabled, clicking the backdrop does not close the modal. The user must dismiss it through the confirm or cancel buttons.

::component-preview{file="SModalConfirm/prevent-close"}
::

## Responsive

By default, `responsive` is `true`. On mobile, the modal is positioned at the bottom of the screen. On larger screens, it is vertically centered. Set `responsive` to `false` to always center the content.

::component-preview{file="SModalConfirm/responsive"}
::

## Custom Content

Use the default slot to replace the `description` prop with custom HTML content.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controls the visibility of the modal |
| `description` | `string` | - | Description text shown in the modal body |
| `title` | `string` | `'Confirm'` | Modal title text |
| `icon` | `FunctionalComponent \| 'primary' \| 'success' \| 'danger' \| 'warning' \| 'info'` | `'danger'` | Icon to display. Use a string for presets or a component for custom icons |
| `confirmText` | `string` | `'Confirm'` | Text for the confirm button |
| `cancelText` | `string` | `'Cancel'` | Text for the cancel button |
| `closable` | `boolean` | `false` | Shows a close button in the header |
| `responsive` | `boolean` | `true` | When true, positions content at the bottom on mobile and centered on desktop |
| `preventClose` | `boolean` | `false` | Prevents the modal from closing when clicking the backdrop |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:open` | `boolean` | Emitted when the modal requests to open or close |
| `confirm` | - | Emitted when the confirm button is clicked |

## Slots

| Name | Description |
|------|-------------|
| `default` | Replaces the description text with custom content |
