# Usage

`SModalCard` combines `SModal` and `SCard` into a single component, giving you a fully styled modal dialog out of the box. It supports titles, descriptions, icons, closable behavior, and action buttons.

Use `v-model:open` or `:open` + `@update:open` to control visibility.

::component-preview{file="SModalCard/basic"}
::

## Sizes

Use the `size` prop to change the card's padding and border radius.

::component-preview{file="SModalCard/sizes"}
::

## With Icon

Use the `icon` prop with a preset string (`'primary'`, `'success'`, `'danger'`, `'warning'`, `'info'`) to display a predefined icon with matching colors.

::component-preview{file="SModalCard/with-icon"}
::

## Icon Type

Use the `iconType` prop to switch between `'solid'` (default) and `'ping'` icon styles.

::component-preview{file="SModalCard/icon-type"}
::

## Closable

Set `closable` to `true` to display a close button in the card header. The close button emits `update:open` with `false`.

::component-preview{file="SModalCard/closable"}
::

## Prevent Close

When `preventClose` is enabled, clicking the backdrop does not close the modal. The user must dismiss it through an explicit action.

::component-preview{file="SModalCard/prevent-close"}
::

## Responsive

By default, `responsive` is `true`. On mobile, the modal is positioned at the bottom of the screen. On larger screens, it is vertically centered. Set `responsive` to `false` to always center the content.

::component-preview{file="SModalCard/responsive"}
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controls the visibility of the modal |
| `title` | `string` | - | Card title text |
| `size` | `'sm' \| 'md'` | `'md'` | Card size (affects padding and border radius) |
| `icon` | `FunctionalComponent \| 'primary' \| 'success' \| 'danger' \| 'warning' \| 'info'` | - | Icon to display. Use a string for presets or a component for custom icons |
| `iconColor` | `'primary' \| 'success' \| 'danger' \| 'warning' \| 'info'` | - | Overrides the icon color independently of the icon variant |
| `iconType` | `'solid' \| 'ping'` | `'solid'` | Icon display style |
| `closable` | `boolean` | - | Shows a close button in the card header |
| `responsive` | `boolean` | `true` | When true, positions content at the bottom on mobile and centered on desktop |
| `preventClose` | `boolean` | `false` | Prevents the modal from closing when clicking the backdrop |
| `actions` | `TAction[]` | - | Array of action buttons rendered in a divided footer |
| `class` | `string` | - | Additional CSS classes applied to the card |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:open` | `boolean` | Emitted when the modal requests to close (backdrop click or close button) |

## Slots

| Name | Description |
|------|-------------|
| `default` | Main card content |
| `description` | Description text below the title |
| `actions` | Action buttons section at the bottom of the card |
