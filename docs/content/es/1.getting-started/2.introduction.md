**Spartan** es el sistema de componentes unificado de PlaceToPay, construido con [Vue 3](https://vuejs.org) y [TailwindCSS](https://tailwindcss.com). Proporciona más de 50 componentes listos para usar en todas las aplicaciones web de la organización.

## ¿Qué es Spartan?

Spartan es una biblioteca de componentes Vue 3 diseñada específicamente para estandarizar la interfaz de usuario en las aplicaciones de PlaceToPay. Su objetivo es garantizar consistencia visual y de comportamiento sin sacrificar flexibilidad.

La biblioteca se distribuye como un paquete npm y se integra nativamente en cualquier proyecto que use Vue 3 y TailwindCSS.

## Tecnologías

Spartan está construido sobre las siguientes tecnologías:

- **[Vue 3](https://vuejs.org)** — Framework de JavaScript para construir interfaces de usuario.
- **[TailwindCSS](https://tailwindcss.com)** — Framework de CSS de utilidades para un estilo rápido y consistente.
- **[TypeScript](https://www.typescriptlang.org)** — Superset de JavaScript con tipado estático.
- **[Vite](https://vitejs.dev)** — Herramienta de compilación moderna y rápida.

## Ecosistema recomendado

Para aprovechar al máximo Spartan, se recomienda usar las siguientes bibliotecas complementarias:

- **[Iconsax Vue](https://placetopay-org.github.io/iconsax-vue/)** — Conjunto de iconos diseñado para integrar con los componentes de Spartan.
- **[Flagicons Vue](https://placetopay-org.github.io/flagicons-vue/)** — Iconos de banderas por país y región.
- **[vue-i18n](https://vue-i18n.intlify.dev/)** — Solución de internacionalización para Vue, necesaria para los componentes con texto integrado.
