# Uso

El componente `SRadioGroup` envuelve el `RadioGroup` de HeadlessUI para proporcionar una interfaz de seleccion tipo tarjeta. Usa `SRadioGroupItem` dentro para definir cada opcion con titulo, descripcion y pie de pagina.

::spartan-component
---
props:
  modelValue: startup
model:
  - modelValue
slots:
  default:
    children:
      - component: SRadioGroupItem
        props:
          value: startup
        slots:
          title: Startup
          description: 12GB / 6 CPUs
          footer: $40/mo
      - component: SRadioGroupItem
        props:
          value: business
        slots:
          title: Business
          description: 16GB / 8 CPUs
          footer: $80/mo
      - component: SRadioGroupItem
        props:
          value: enterprise
        slots:
          title: Enterprise
          description: 32GB / 12 CPUs
          footer: $160/mo
---
::

## Uso Simple

Puedes usar `SRadioGroupItem` solo con el slot de titulo para tarjetas de seleccion mas simples.

::spartan-component
---
props:
  modelValue: free
model:
  - modelValue
slots:
  default:
    children:
      - component: SRadioGroupItem
        props:
          value: free
        slots:
          title: Free
      - component: SRadioGroupItem
        props:
          value: pro
        slots:
          title: Pro
---
::

## Props de SRadioGroup

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `string` | - | El valor actualmente seleccionado. Requerido. |

## Eventos de SRadioGroup

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `string` | Se emite cuando la opcion seleccionada cambia. |

## Slots de SRadioGroup

| Nombre | Descripcion |
|--------|-------------|
| `default` | Contenedor para los componentes `SRadioGroupItem`. |

## Props de SRadioGroupItem

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `value` | `string` | - | El valor que esta opcion representa. Requerido. |

## Slots de SRadioGroupItem

| Nombre | Descripcion |
|--------|-------------|
| `title` | La etiqueta principal de la opcion. |
| `description` | Texto descriptivo adicional debajo del titulo. |
| `footer` | Contenido del pie, tipicamente usado para precios o metadatos. |
