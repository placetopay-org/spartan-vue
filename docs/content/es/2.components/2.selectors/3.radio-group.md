# Uso

El componente `SRadioGroup` envuelve el `RadioGroup` de HeadlessUI para proporcionar una interfaz de selección tipo tarjeta. Usa `SRadioGroupItem` dentro para definir cada opción con título, descripción y pie de página.

::component-preview{file="SRadioGroup/basic"}
::

## Uso Simple

Puedes usar `SRadioGroupItem` solo con el slot de título para tarjetas de selección más simples.

::component-preview{file="SRadioGroup/simple"}
::

## Props de SRadioGroup

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `modelValue` | `string` | - | El valor actualmente seleccionado. Requerido. |

## Eventos de SRadioGroup

| Evento | Payload | Descripción |
|--------|---------|-------------|
| `update:modelValue` | `string` | Se emite cuando la opción seleccionada cambia. |

## Slots de SRadioGroup

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenedor para los componentes `SRadioGroupItem`. |

## Props de SRadioGroupItem

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `value` | `string` | - | El valor que esta opción representa. Requerido. |

## Slots de SRadioGroupItem

| Nombre | Descripción |
|--------|-------------|
| `title` | La etiqueta principal de la opción. |
| `description` | Texto descriptivo adicional debajo del título. |
| `footer` | Contenido del pie, típicamente usado para precios o metadatos. |
