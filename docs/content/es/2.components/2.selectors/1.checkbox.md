# Uso

El componente `SCheckbox` proporciona un input de checkbox estilizado con slots opcionales de etiqueta y descripcion. Soporta tanto seleccion booleana individual como seleccion grupal basada en arrays mediante `v-model`.

::spartan-component
---
props:
  disabled: false
  reverse: false
  inline: false
slots:
  default: Accept terms
  description: You agree to our terms and conditions.
ignore:
  - id
  - name
  - value
  - modelValue
  - class
---
::

## Deshabilitado

Usa la prop `disabled` para prevenir la interaccion del usuario con el checkbox. El componente aparecera con opacidad reducida.

::spartan-component
---
props:
  disabled: true
slots:
  default: Disabled option
ignore:
  - id
  - name
  - value
  - modelValue
  - reverse
  - inline
  - class
---
::

## Invertido

Usa la prop `reverse` para colocar el checkbox en el lado derecho con la etiqueta a la izquierda, usando un layout justify-between.

::spartan-component
---
props:
  reverse: true
slots:
  default: Reversed checkbox
  description: The checkbox appears on the right side.
ignore:
  - id
  - name
  - value
  - modelValue
  - disabled
  - inline
  - class
---
::

## En Linea

Usa la prop `inline` para renderizar la etiqueta en linea con el texto de descripcion en lugar de encima de el.

::spartan-component
---
props:
  inline: true
slots:
  default: Remember me
  description: Save my login details for next time.
ignore:
  - id
  - name
  - value
  - modelValue
  - disabled
  - reverse
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `boolean \| string[]` | - | El valor vinculado. Usa `boolean` para checkboxes individuales o `string[]` para grupos. |
| `value` | `string` | - | El identificador de valor cuando se usa en grupo con un `modelValue` de tipo array. |
| `disabled` | `boolean` | `false` | Deshabilita el checkbox y reduce la opacidad. |
| `reverse` | `boolean` | `false` | Coloca el checkbox en el lado derecho de la etiqueta. |
| `inline` | `boolean` | `false` | Renderiza la etiqueta en linea con la descripcion. |
| `name` | `string` | - | El atributo HTML `name` del input. |
| `id` | `string` | auto-generado | El atributo HTML `id`. Se genera automaticamente si no se proporciona. |

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `boolean \| string[]` | Se emite cuando el estado del checkbox cambia. |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | El texto de la etiqueta del checkbox. |
| `description` | Texto de descripcion adicional mostrado debajo o al lado de la etiqueta. |
