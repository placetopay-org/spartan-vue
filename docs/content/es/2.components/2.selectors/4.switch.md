# Uso

El componente `SSwitch` proporciona un interruptor toggle estilizado construido sobre HeadlessUI. Soporta etiquetas, descripciones, iconos y modo pasivo.

::spartan-component
---
props:
  modelValue: false
  reverse: false
  passive: false
slots:
  default: Enable notifications
  description: Receive alerts when something happens.
model:
  - modelValue
ignore:
  - icon
  - iconOff
  - iconOn
  - class
---
::

## Invertido

Usa la prop `reverse` para colocar el switch en el lado derecho con la etiqueta a la izquierda.

::spartan-component
---
props:
  modelValue: false
  reverse: true
slots:
  default: Dark mode
  description: Switch to dark theme.
model:
  - modelValue
ignore:
  - icon
  - iconOff
  - iconOn
  - passive
  - class
---
::

## Pasivo

Cuando `passive` esta habilitado, hacer clic en la etiqueta o descripcion no cambia el switch. Solo hacer clic en el switch mismo cambia el valor.

::spartan-component
---
props:
  modelValue: false
  passive: true
slots:
  default: Passive switch
  description: Only the switch toggles, not the label.
model:
  - modelValue
ignore:
  - icon
  - iconOff
  - iconOn
  - reverse
  - class
---
::

## Con Icono

Establece la prop `icon` en `true` para mostrar iconos de check/cruz por defecto dentro del boton del switch.

::spartan-component
---
props:
  modelValue: false
  icon: true
slots:
  default: With default icons
model:
  - modelValue
ignore:
  - iconOff
  - iconOn
  - reverse
  - passive
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `boolean` | - | El estado actual del toggle. Requerido. |
| `reverse` | `boolean` | `false` | Coloca el switch en el lado derecho de la etiqueta. |
| `passive` | `boolean` | `false` | Previene que los clics en la etiqueta/descripcion cambien el switch. |
| `icon` | `boolean \| FunctionalComponent` | - | Muestra iconos por defecto cuando es `true`, o un componente de icono personalizado para ambos estados. |
| `iconOn` | `FunctionalComponent` | - | Componente de icono personalizado para el estado ON. |
| `iconOff` | `FunctionalComponent` | - | Componente de icono personalizado para el estado OFF. |

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `boolean` | Se emite cuando el switch es alternado. |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | El texto de la etiqueta del switch. |
| `description` | Texto de descripcion adicional debajo de la etiqueta. |
