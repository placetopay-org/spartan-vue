# Uso

El componente `SRadio` proporciona un input de radio estilizado con slots opcionales de etiqueta y descripcion. Usa la prop `name` para agrupar radios y `value` para identificar cada opcion.

::spartan-component
---
props:
  value: option1
  name: example
  disabled: false
  reverse: false
  inline: false
slots:
  default: Option 1
  description: Select this option to continue.
ignore:
  - id
  - modelValue
  - class
---
::

## Deshabilitado

Usa la prop `disabled` para prevenir la interaccion del usuario con el radio. El componente aparecera con opacidad reducida.

::spartan-component
---
props:
  value: disabled-option
  name: disabled-example
  disabled: true
slots:
  default: Disabled option
ignore:
  - id
  - modelValue
  - reverse
  - inline
  - class
---
::

## Invertido

Usa la prop `reverse` para colocar el radio en el lado derecho con la etiqueta a la izquierda.

::spartan-component
---
props:
  value: reversed
  name: reverse-example
  reverse: true
slots:
  default: Reversed radio
  description: The radio appears on the right side.
ignore:
  - id
  - modelValue
  - disabled
  - inline
  - class
---
::

## En Linea

Usa la prop `inline` para renderizar la etiqueta en linea con el texto de descripcion en lugar de encima de el.

::spartan-component
---
props:
  value: inline-option
  name: inline-example
  inline: true
slots:
  default: Inline radio
  description: The label appears inline with the description.
ignore:
  - id
  - modelValue
  - disabled
  - reverse
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `boolean \| string` | - | El valor vinculado para el grupo de radios. Requerido. |
| `value` | `string` | - | El valor que este radio representa. Requerido. |
| `disabled` | `boolean` | `false` | Deshabilita el radio y reduce la opacidad. |
| `reverse` | `boolean` | `false` | Coloca el radio en el lado derecho de la etiqueta. |
| `inline` | `boolean` | `false` | Renderiza la etiqueta en linea con la descripcion. |
| `name` | `string` | - | El atributo HTML `name` para agrupar radios. |
| `id` | `string` | auto-generado | El atributo HTML `id`. Se genera automaticamente si no se proporciona. |

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `boolean \| string` | Se emite cuando el radio es seleccionado. |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | El texto de la etiqueta del radio. |
| `description` | Texto de descripcion adicional mostrado debajo o al lado de la etiqueta. |
