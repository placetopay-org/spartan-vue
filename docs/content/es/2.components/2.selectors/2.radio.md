# Uso

El componente `SRadio` proporciona un input de radio estilizado con slots opcionales de etiqueta y descripción. Usa la prop `name` para agrupar radios y `value` para identificar cada opción.

::component-preview{file="SRadio/basic"}
::

## Deshabilitado

Usa la prop `disabled` para prevenir la interacción del usuario con el radio. El componente aparecerá con opacidad reducida.

::component-preview{file="SRadio/disabled"}
::

## Invertido

Usa la prop `reverse` para colocar el radio en el lado derecho con la etiqueta a la izquierda.

::component-preview{file="SRadio/reverse"}
::

## En Línea

Usa la prop `inline` para renderizar la etiqueta en línea con el texto de descripción en lugar de encima de él.

::component-preview{file="SRadio/inline"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `modelValue` | `boolean \| string` | - | El valor vinculado para el grupo de radios. Requerido. |
| `value` | `string` | - | El valor que este radio representa. Requerido. |
| `disabled` | `boolean` | `false` | Deshabilita el radio y reduce la opacidad. |
| `reverse` | `boolean` | `false` | Coloca el radio en el lado derecho de la etiqueta. |
| `inline` | `boolean` | `false` | Renderiza la etiqueta en línea con la descripción. |
| `name` | `string` | - | El atributo HTML `name` para agrupar radios. |
| `id` | `string` | auto-generado | El atributo HTML `id`. Se genera automáticamente si no se proporciona. |

## Eventos

| Evento | Payload | Descripción |
|--------|---------|-------------|
| `update:modelValue` | `boolean \| string` | Se emite cuando el radio es seleccionado. |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | El texto de la etiqueta del radio. |
| `description` | Texto de descripción adicional mostrado debajo o al lado de la etiqueta. |
