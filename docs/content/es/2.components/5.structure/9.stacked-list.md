# Uso

El componente `SStackedList` renderiza una lista estilizada con divisores. Usa hijos `SStackedListItem` para definir cada elemento.

::component-preview{file="SStackedList/basic"}
::

## Menu Desplegable

Usa el slot `dropdown` en `SStackedListItem` para agregar un menu de opciones con elementos `SDropdownItem`.

::component-preview{file="SStackedList/dropdown"}
::

## Props

### SStackedList

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `class` | `string` | - | Clases CSS adicionales para el contenedor de la lista |

### SStackedListItem

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `class` | `string` | - | Clases CSS adicionales para el contenido del elemento |

## Slots

### SStackedList

| Nombre | Descripcion |
|--------|-------------|
| `default` | Componentes `SStackedListItem` |

### SStackedListItem

| Nombre | Descripcion |
|--------|-------------|
| `default` | El contenido del elemento |
| `dropdown` | Elementos del menu desplegable (muestra un icono de opciones) |
