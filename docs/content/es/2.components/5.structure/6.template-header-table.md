# Uso

El componente `STemplateHeaderTable` renderiza un encabezado con titulo e input de busqueda. Usa el slot `table` para colocar el contenido de tu tabla debajo.

::component-preview{file="STemplateHeaderTable/basic"}
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `title` | `string` | - | El texto del encabezado mostrado en la cabecera |

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `search` | `string` | Se emite cuando el usuario hace clic en el boton de busqueda o presiona Enter |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `table` | El contenido de la tabla a mostrar debajo del encabezado |
