# Uso

Utilice el slot predeterminado para configurar la etiqueta del badge.

::component-code
---
prettier: true
slots:
  default: Badge
---
::

### Color

Usa la prop `color` para cambiar el color del Badge.

::component-code
---
prettier: true
props:
  color: neutral
slots:
  default: Badge
  other: Color neutral
---
::
