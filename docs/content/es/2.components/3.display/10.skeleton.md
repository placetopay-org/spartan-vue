# Uso

El componente `SSkeleton` proporciona un placeholder animado para indicar que el contenido está cargando. Renderiza una animación shimmer y puede personalizarse mediante la prop `class`.

::spartan-component
---
props:
  class: 'h-6 w-full rounded-md'
---
::

## Tamaños

Personaliza la altura y el ancho usando clases de Tailwind.

::spartan-component
---
props:
  class: 'h-4 w-3/4 rounded-md'
---
::

::spartan-component
---
props:
  class: 'h-8 w-1/2 rounded-md'
---
::

::spartan-component
---
props:
  class: 'h-12 w-full rounded-lg'
---
::

## Composición

Combina múltiples skeletons para construir placeholders de carga para layouts complejos.

```vue
<template>
  <div class="flex items-center gap-4">
    <SSkeleton class="h-12 w-12 rounded-full" />
    <div class="flex-1 space-y-2">
      <SSkeleton class="h-4 w-3/4" />
      <SSkeleton class="h-4 w-1/2" />
    </div>
  </div>
</template>
```

```vue
<template>
  <div class="space-y-3">
    <SSkeleton class="h-40 w-full rounded-lg" />
    <SSkeleton class="h-4 w-3/4" />
    <SSkeleton class="h-4 w-1/2" />
    <SSkeleton class="h-4 w-5/6" />
  </div>
</template>
```

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `class` | `string` | - | Clases CSS para personalizar tamaño, forma y borde. Se mezclan con las clases por defecto (`h-6 w-full rounded-md`) |
