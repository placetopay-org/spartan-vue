# Uso

El componente SCaption muestra texto de ayuda o error pequeño con soporte para modo oscuro, comúnmente utilizado debajo de inputs de formulario para proporcionar contexto adicional o retroalimentación de validación.

::component-preview{file="SCaption/basic"}
::

## Variantes

SCaption admite dos variantes visuales que determinan el color del texto y el significado semántico. Ambas variantes se adaptan automáticamente al modo oscuro.

::component-preview{file="SCaption/variants"}
::

## Con Contenido Personalizado

Puedes usar el slot default para proporcionar contenido personalizado en lugar de usar la prop `text`. Esto es útil cuando necesitas texto formateado o múltiples elementos.

::component-preview{file="SCaption/customContent"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `variant` | `'error' \| 'info'` | `'error'` | La variante visual del caption que determina el color del texto |
| `text` | `string` | - | El contenido de texto a mostrar en el caption |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido personalizado que reemplaza la prop text |
