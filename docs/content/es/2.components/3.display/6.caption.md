# Uso

El componente SCaption muestra texto de ayuda o error pequeño, comúnmente utilizado debajo de inputs de formulario para proporcionar contexto adicional o retroalimentación de validación.

::spartan-component
---
items: { variant: ['error', 'info'] }
props:
  variant: error
  text: This field is required
ignore:
  - class
---
::

## Variantes

SCaption admite dos variantes visuales que determinan el color del texto y el significado semántico.

::spartan-component
---
items: { variant: ['error', 'info'] }
props:
  variant: error
  text: Error message displayed in red
ignore:
  - class
---
::

## Con Contenido Personalizado

Puedes usar el slot default para proporcionar contenido personalizado en lugar de usar la prop `text`. Esto es útil cuando necesitas texto formateado o múltiples elementos.

::spartan-component
---
props:
  variant: info
slots:
  default: Additional information can be provided here
ignore:
  - text
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `variant` | `'error' \| 'info'` | `'error'` | La variante visual del caption que determina el color del texto |
| `text` | `string` | - | El contenido de texto a mostrar en el caption |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | Contenido personalizado que reemplaza la prop text |
