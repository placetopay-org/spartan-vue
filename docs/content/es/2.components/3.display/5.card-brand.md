# Uso

El componente `SCardBrand` renderiza logos de metodos de pago y marcas de tarjetas. Soporta una amplia variedad de marcas incluyendo Visa, Mastercard, Amex, PSE y muchas mas.

::spartan-component
---
items:
  name:
    - visa
    - master
    - amex
    - diners
    - discover
    - pse
    - paypal
    - cash
    - debit
    - elo
    - jcb
    - maestro
props:
  name: visa
  size: 48
ignore:
  - class
---
::

## Tamano

Usa la prop `size` para controlar el ancho y alto del icono de la marca en pixeles.

::spartan-component
---
props:
  name: visa
  size: 64
ignore:
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `name` | `'visa' \| 'master' \| 'amex' \| ...` | - | El nombre de la marca a mostrar. Ver la lista completa arriba. |
| `size` | `number` | - | Ancho y alto del icono SVG en pixeles. |
| `class` | `string` | - | Clases CSS adicionales para aplicar al elemento SVG. |
