# Uso

El componente SLink renderiza una etiqueta anchor estilizada con efectos de hover incorporados y comportamiento de subrayado.

::spartan-component
---
items: { target: ['_self', '_blank', '_parent', '_top'] }
props:
  href: https://example.com
  target: _self
slots:
  default: Click here to visit
ignore:
  - class
---
::

## Target

La prop `target` controla dónde se abrirá el documento vinculado. Usa `_blank` para abrir enlaces en una nueva pestaña, lo cual es común para enlaces externos.

::spartan-component
---
items: { target: ['_self', '_blank', '_parent', '_top'] }
props:
  href: https://example.com
  target: _blank
slots:
  default: Open in new tab
ignore:
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `href` | `string` | - | La URL a la que apunta el enlace (requerido) |
| `target` | `'_blank' \| '_self' \| '_parent' \| '_top'` | - | Especifica dónde abrir el documento vinculado |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | El contenido de texto clicable del enlace |
