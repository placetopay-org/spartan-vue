# Uso

El componente SPageTitle renderiza un elemento h1 con estilos de texto predefinidos (tamaño 2xl, peso semibold, color gray-900).

::spartan-component
---
props: {}
slots:
  default: Dashboard Overview
ignore:
  - class
---
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `class` | `string` | - | Clases CSS adicionales para personalizar la apariencia |

## Slots

| Nombre | Descripcion |
|--------|-------------|
| `default` | El contenido de texto del título de página |
