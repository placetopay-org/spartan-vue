# Uso

El componente SPlacetopayLogo renderiza el logo de la marca Placetopay con soporte para diferentes modos visuales y dimensiones personalizables.

::component-preview{file="SPlacetopayLogo/basic"}
::

## Modos

Elige entre tres modos de color dependiendo de tu fondo: base (gris), dark (blanco) o blackAndWhite.

::component-preview{file="SPlacetopayLogo/modes"}
::

## Tamaño Predefinido

Usa la prop `size` para dimensionar rápidamente. El tamaño "md" establece el ancho en 202px automáticamente mientras mantiene la relación de aspecto.

::component-preview{file="SPlacetopayLogo/size"}
::

## Dimensiones Personalizadas

Establece dimensiones personalizadas usando las props `width` y `height`. El componente mantiene una relación de aspecto de 202:45 por defecto.

::component-preview{file="SPlacetopayLogo/custom-dimensions"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `mode` | `'base' \| 'dark' \| 'blackAndWhite'` | `'base'` | El modo de color del logo |
| `size` | `'md'` | - | Tamaño predefinido (md = 202px de ancho) |
| `width` | `number` | - | Ancho personalizado en píxeles (sobrescribe size) |
| `height` | `number` | - | Alto personalizado en píxeles |
| `class` | `string` | - | Clases CSS adicionales |
