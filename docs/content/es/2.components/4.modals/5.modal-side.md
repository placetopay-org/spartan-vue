# Uso

`SModalSide` es un panel modal de altura completa que puede deslizarse desde el lado izquierdo o derecho de la pantalla. Extiende el concepto de `SModalLeft` añadiendo una prop `side` configurable.

Controla la visibilidad con la prop `:open` y maneja el cierre mediante los eventos `@close` y `@backdrop-click`.

::component-preview{file="SModalSide/basic"}
::

## Lado

Usa la prop `side` para controlar desde qué borde se desliza el panel. Por defecto es `'left'`.

::component-preview{file="SModalSide/side"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | Controla la visibilidad del modal |
| `side` | `'left' \| 'right'` | `'left'` | Lado desde el cual se desliza el modal |
| `class` | `string` | - | Clases CSS adicionales para el elemento raíz |
| `backdropClass` | `string` | `''` | Clases CSS adicionales para el fondo del overlay |
| `breakpoint` | `'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| number` | - | Punto de quiebre por encima del cual el modal se oculta. Útil para diseños responsivos donde el modal solo debe aparecer en pantallas pequeñas |

## Eventos

| Evento | Payload | Descripción |
|--------|---------|-------------|
| `close` | - | Se emite cuando se hace clic en el botón de cierre |
| `backdropClick` | - | Se emite cuando se hace clic en el fondo |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido del panel lateral |
