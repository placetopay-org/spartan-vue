# Uso

El componente `STextArea` renderiza un elemento `<textarea>` nativo. Usa `v-model` para vincular el valor del texto.

::component-preview{file="STextarea/basic"}
::

## Error

Establece `error` en `true` para mostrar el textarea con estilo de error.

::component-preview{file="STextarea/error"}
::

## Deshabilitado

Establece `disabled` en `true` para evitar la interaccion del usuario.

::component-preview{file="STextarea/disabled"}
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `string` | - | El valor del textarea (v-model) |
| `error` | `boolean` | `false` | Muestra estilos de error |
| `disabled` | `boolean` | `false` | Deshabilita el textarea |
| `class` | `string` | - | Clases CSS adicionales |

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `string` | Se emite cuando el valor del textarea cambia |
