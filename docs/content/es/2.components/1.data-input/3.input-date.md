# Uso

El componente `SInputDate` renderiza un selector de fecha. Usa `v-model` para vincular el valor de la fecha seleccionada.

::component-preview{file="SInputDate/basic"}
::

## Formato

Usa la prop `modelType` para controlar la cadena de formato de fecha usada para el valor del modelo.

::component-preview{file="SInputDate/format"}
::

## Error

Establece `error` en `true` para mostrar el input con estilo de error.

::component-preview{file="SInputDate/error"}
::

## Ocultar Icono

Establece `hideInputIcon` en `true` para ocultar el icono de calendario dentro del input.

::component-preview{file="SInputDate/hide-icon"}
::

## Props

| Prop | Tipo | Default | Descripcion |
|------|------|---------|-------------|
| `modelValue` | `string \| string[] \| null` | - | El valor de la fecha seleccionada (v-model) |
| `placeholder` | `string` | - | Texto placeholder para el input |
| `error` | `boolean` | `false` | Muestra estilos de error |
| `hideInputIcon` | `boolean` | `false` | Oculta el icono de calendario |
| `id` | `string` | - | Atributo HTML id para el input |
| `modelType` | `string` | - | Cadena de formato de fecha (ej. `'dd-MM-yyyy'`) |
| `format` | `string` | - | Formato de visualizacion de la fecha |
| `class` | `string` | - | Clases CSS adicionales |

El componente tambien acepta todas las props de [VueDatePicker](https://vue3datepicker.com/).

## Eventos

| Evento | Payload | Descripcion |
|--------|---------|-------------|
| `update:modelValue` | `string \| string[] \| null` | Se emite cuando la fecha seleccionada cambia |
