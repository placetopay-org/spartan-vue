# Uso

El componente `SLabel` es un elemento de etiqueta estilizado para inputs de formulario, proporcionando tipografía consistente y opciones de accesibilidad. Usa la prop `for` para asociarlo con un input de formulario por su ID.

::spartan-component
---
props:
  srOnly: false
slots:
  default: Dirección de correo
---
::

## Solo para lectores de pantalla

Usa la prop `srOnly` para ocultar visualmente la etiqueta mientras la mantiene accesible para lectores de pantalla.

::spartan-component
---
items: { srOnly: [false, true] }
props:
  srOnly: false
slots:
  default: Etiqueta para Lector de Pantalla
---
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `for` | `string` | - | El ID del elemento input con el que esta etiqueta está asociada |
| `srOnly` | `boolean` | `false` | Ocultar visualmente la etiqueta (solo lector de pantalla) |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido de texto de la etiqueta |
