# Uso

El componente `SCopy` permite a los usuarios copiar texto al portapapeles con un solo clic. Muestra un icono de copiar que cambia a un estado de éxito y muestra un tooltip cuando el texto es copiado.

::spartan-component
---
props:
  value: 'Hola Mundo'
slots:
  default: Hola Mundo
---
::

## Con prop value

Usa la prop `value` para especificar el texto a copiar. Si no se proporciona, el componente copiará el contenido de texto del slot por defecto.

::spartan-component
---
props:
  value: 'Texto personalizado para copiar'
slots:
  default: Clic para copiar texto personalizado
---
::

## Sin prop value

Cuando no se proporciona la prop `value`, el componente copia automáticamente el contenido de texto renderizado dentro del slot por defecto.

::spartan-component
---
slots:
  default: Este texto será copiado
---
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `value` | `string` | - | Texto a copiar al portapapeles. Si no se proporciona, copia el contenido del slot |
| `class` | `string` | - | Clases CSS adicionales |

## Eventos

| Evento | Payload | Descripción |
|--------|---------|-------------|
| `copied` | `string` | Se emite cuando el texto es copiado, con el valor copiado |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido mostrado junto al icono de copiar. También se usa como texto de respaldo a copiar si `value` no se proporciona |
