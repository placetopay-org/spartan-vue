# Uso

El componente `SSectionDescription` renderiza texto descriptivo estilizado. Usa la prop `as` para controlar qué elemento HTML se utiliza.

::spartan-component
---
props:
  as: p
slots:
  default: Esta es una descripción de sección que proporciona contexto adicional sobre el contenido.
---
::

## Elemento HTML

Usa la prop `as` para renderizar un elemento HTML diferente. Por defecto es `p`.

::spartan-component
---
items: { as: ['p', 'span', 'div'] }
props:
  as: p
slots:
  default: Descripción renderizada con un elemento diferente.
---
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `as` | `string` | `'p'` | El elemento HTML a renderizar |
| `class` | `string` | - | Clases CSS adicionales |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido de texto de la descripción |
