# Uso

El componente `SSectionTitle` renderiza un elemento de encabezado estilizado con soporte para modo oscuro. Usa la prop `as` para controlar qué etiqueta HTML de encabezado se utiliza.

::component-preview{file="SSectionTitle/basic"}
::

## Niveles de encabezado

Usa la prop `as` para renderizar diferentes elementos HTML de encabezado (`h1` a `h6`).

::component-preview{file="SSectionTitle/heading-levels"}
::

## Estilos personalizados

Pasa una prop `class` para sobrescribir o extender los estilos predeterminados. Las clases se combinan inteligentemente usando `tailwind-merge`.

::component-preview{file="SSectionTitle/custom-styles"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `as` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6'` | `'h3'` | El elemento HTML de encabezado a renderizar |
| `class` | `string` | - | Clases CSS adicionales para personalizar la apariencia |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido de texto del encabezado |
