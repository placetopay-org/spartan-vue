# Uso

El componente `SSectionDescription` renderiza texto descriptivo estilizado con soporte para modo oscuro. Usa la prop `as` para controlar qué elemento HTML se utiliza.

::component-preview{file="SSectionDescription/basic"}
::

## Elemento HTML

Usa la prop `as` para renderizar un elemento HTML diferente. Por defecto es `p`.

::component-preview{file="SSectionDescription/element"}
::

## Estilos personalizados

Pasa una prop `class` para sobrescribir o extender los estilos predeterminados. Las clases se combinan inteligentemente usando `tailwind-merge`.

::component-preview{file="SSectionDescription/custom-styles"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `as` | `string` | `'p'` | El elemento HTML a renderizar |
| `class` | `string` | - | Clases CSS adicionales para personalizar la apariencia |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | Contenido de texto de la descripción |
