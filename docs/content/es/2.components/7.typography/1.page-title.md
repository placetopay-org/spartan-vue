# Uso

El componente SPageTitle renderiza un elemento `h1` semántico con estilos de texto predefinidos (tamaño 2xl, peso semibold) y soporte para modo oscuro. Úsalo para mostrar el encabezado principal de cualquier página con tipografía consistente.

::component-preview{file="SPageTitle/basic"}
::

## Clase personalizada

Puedes sobrescribir los estilos por defecto usando la prop `class`. El componente usa `tailwind-merge` para fusionar inteligentemente tus clases personalizadas con las predeterminadas.

::component-preview{file="SPageTitle/custom-class"}
::

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `class` | `HTMLAttributes['class']` | - | Clases CSS adicionales para personalizar la apariencia |

## Slots

| Nombre | Descripción |
|--------|-------------|
| `default` | El contenido de texto del título de página |
