# Migration Guide: Spartan Vue 2.x to 3.x

Migrating from Spartan Vue 2.x (Tailwind CSS 3) to Spartan Vue 3.x (Tailwind CSS 4).

## Quick Overview

Spartan Vue 3.x introduces full support for Tailwind CSS 4 with a CSS-native configuration approach.

### Migration Steps Summary

1. **Remove** Spartan Vue plugin from `tailwind.config.js`
2. **Run** `npx @tailwindcss/upgrade` to migrate to Tailwind 4
3. **Remove** PostCSS configuration (if applicable)
4. **Install** `@tailwindcss/vite` plugin (if using Vite)
5. **Update** to Spartan Vue 3.x: `npm install @placetopay/spartan-vue@latest`
6. **Move** CSS imports from JavaScript to your main CSS file
7. **Update** color imports API (if using Tailwind colors in JS)
8. **Test** your application

## Key Changes (Breaking Changes)

- No more JavaScript `tailwind.config.js` - now uses CSS-based configuration
- CSS imports moved from JavaScript to CSS files using `@import`
- Tailwind color imports API changed from named exports to default export
- Single `styles.css` import replaces multiple imports

## Resources

- Full Migration Guide: https://github.com/placetopay-org/spartan-vue/blob/main/MIGRATION.md
- Tailwind CSS v4 Upgrade Guide: https://tailwindcss.com/docs/upgrade-guide
- Spartan Vue Documentation: https://develop--646e732a14dfaa707ad59b33.chromatic.com/
- Report Issues: https://github.com/placetopay-org/spartan-vue/issues

export const buildSourceBinding = (bindings: TBindings, map?: string) => (args: any) => {
    let result = '';

    if (bindings.prop) {
        Object.keys(bindings.prop).forEach((prop) => {
            if (args[prop] !== undefined && args[prop] !== '' && args[prop] !== bindings.prop![prop])
                result += `${prop}="${args[prop]}" `;
        });
    }

    if (bindings.custom?.icon) {
        if (args[getBinding('icon', map)]) result += `:icon="${args[getBinding('icon', map)]}" `;
    }

    if (bindings.check) {
        bindings.check.forEach((binding) => {
            if (args[getBinding(binding, map)]) result += `${binding} `;
        });
    }

    if (bindings.emit) {
        bindings.emit.forEach((emit) => {
            if (args[emit]) result += `@${emit}="${args[emit].replace(/ /g, '')}" `;
        });
    }

    return result.trim();
};
